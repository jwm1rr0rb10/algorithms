# Генерация кода Грея (Gray Code)

## Описание задачи

Сгенерируйте последовательность n-битных кодов Грея (Gray code).  
Код Грея — это такая последовательность двоичных чисел длины n, в которой каждое следующее число отличается от предыдущего ровно одним битом.

**Пример для n = 2:**
- 00
- 01
- 11
- 10

**Пример для n = 3:**
- 000
- 001
- 011
- 010
- 110
- 111
- 101
- 100

---

## Идея и подход

- Для любого числа `i` его n-битный код Грея вычисляется по формуле:  
  `gray = i ^ (i >> 1)`  
  где `^` — XOR (побитовое исключающее ИЛИ), `>>` — сдвиг вправо.
- Чтобы получить всю последовательность, перебираем все числа от 0 до `2^n - 1` и считаем для каждого код Грея.

---

## Пример на Python

```python
def gray_code_sequence(n):
    res = []
    for i in range(1 << n):  # 1 << n == 2^n
        res.append(i ^ (i >> 1))
    return res

# Пример использования:
print(gray_code_sequence(2))  # [0, 1, 3, 2]
print(gray_code_sequence(3))  # [0, 1, 3, 2, 6, 7, 5, 4]

# Для вывода в двоичном виде с ведущими нулями:
n = 3
for x in gray_code_sequence(n):
    print(f"{x:03b}")
```

---

## Как это работает

- Каждый следующий элемент отличается только одним битом от предыдущего.
- Это свойство полезно для минимизации ошибок при переходах состояний (например, в энкодерах, при передаче данных).

## Применение

- Минимизация ошибок при преобразовании аналогового сигнала в цифровой (АЦП)
- Работа с энкодерами
- Перебор подмножеств с минимальным изменением
- Проектирование цифровых устройств

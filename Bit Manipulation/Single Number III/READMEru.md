# Single Number III (Два уникальных числа)

## Описание задачи

Дан массив целых чисел, в котором ровно два элемента встречаются по одному разу, а все остальные — ровно по два. Найдите эти два уникальных числа. Порядок не важен.

---

## Подход к решению

Оптимальное решение использует побитовые операции:

1. **XOR всех чисел:**  
   XOR всех элементов массива даст `a ^ b`, где `a` и `b` — два уникальных числа (все остальные пары взаимно уничтожаются).

2. **Найти отличающий бит:**  
   Находим любой установленный бит в результате XOR (например, самый правый). В этом бите `a` и `b` различаются.

3. **Разделить и снова XOR:**  
   Делим все числа на две группы: с этим битом и без него. В каждой группе XOR даст по одному уникальному числу.

---

## Пример на Python

```python
def singleNumber(nums):
    xor = 0
    for num in nums:
        xor ^= num
    # Находим самый правый отличающийся бит
    diff = xor & -xor
    a = b = 0
    for num in nums:
        if num & diff:
            a ^= num
        else:
            b ^= num
    return [a, b]

# Пример:
nums = [1,2,1,3,2,5]
print(singleNumber(nums))  # [3, 5] (порядок не важен)
```

---

## Как это работает

- XOR всех чисел даёт `a ^ b`
- Отличающийся бит показывает, где числа различаются
- Деление на группы и повторный XOR выделяет оба уникальных числа

---

## Применение

- Поиск двух уникальных элементов среди парных дубликатов
- Эффективные битовые трюки для анализа данных и проверки ошибок

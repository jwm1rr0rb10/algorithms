# Single Number II (Уникальное число II)

## Описание задачи

Дан массив целых чисел, где каждый элемент встречается ровно три раза, кроме одного, который встречается один раз. Найдите это единственное число.

---

## Подход к решению

Оптимальное решение — использовать побитовые операции, что даёт сложность O(n) по времени и O(1) по памяти.

**Идея:**  
Для каждой битовой позиции (от 0 до 31) считаем, сколько чисел имеют этот бит установленным. Если сумма по модулю 3 не равна 0 — этот бит принадлежит уникальному числу.

---

## Пример на Python

```python
def singleNumber(nums):
    result = 0
    for i in range(32):
        count = 0
        for num in nums:
            if (num >> i) & 1:
                count += 1
        if count % 3:
            # Учёт отрицательных чисел для 32 бит
            if i == 31:
                result -= (1 << 31)
            else:
                result |= (1 << i)
    return result

# Пример:
nums = [2,2,3,2]
print(singleNumber(nums))  # 3
```

---

## Как это работает

- Для каждой битовой позиции считаем количество единиц среди всех чисел.
- Если остаток от деления на 3 не равен 0 — этот бит есть в искомом числе.
- Для поддержки отрицательных чисел корректируем знаковый бит (31-й).

---

## Применение

- Поиск уникального значения среди чисел, встречающихся k раз.
- Битовые задачи на собеседованиях.
- Проверка целостности данных.

# Установить i-й бит

## Описание задачи

Дано целое число `n` и позиция бита `i` (нумерация с 0 от младшего бита). Нужно установить (сделать равным 1) i-й бит в числе `n`. Остальные биты должны остаться без изменений.

---

## Подход к решению

- Используем побитовое ИЛИ (OR) с маской, где только i-й бит равен 1.
- Маска создаётся сдвигом `1` влево на `i` позиций: `1 << i`.
- Операция `n | (1 << i)` устанавливает i-й бит в `n`, остальные не меняет.

---

## Пример на Python

```python
def set_ith_bit(n, i):
    return n | (1 << i)

# Примеры:
print(bin(set_ith_bit(0b1010, 1)))  # 0b1010 | 0b0010 => 0b1010 (бит уже был установлен)
print(bin(set_ith_bit(0b1010, 2)))  # 0b1010 | 0b0100 => 0b1110 (бит был 0, стал 1)
print(bin(set_ith_bit(0b0, 4)))     # 0b0000 | 0b10000 => 0b10000
```

---

## Как это работает

- `1 << i` создаёт маску с единственным установленным битом на позиции i.
- Побитовое ИЛИ (`|`) устанавливает этот бит в `n`, остальные не изменяются.

---

## Применение

- Установка флагов в битовых масках.
- Управление правами, настройками, состояниями через битовые поля.
- Быстрое изменение отдельных признаков в числовых данных.

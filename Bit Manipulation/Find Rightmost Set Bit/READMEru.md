# Поиск самого младшего установленного бита

## Описание задачи

Дано целое число `n`.  
Найдите позицию (индекс) самого правого (младшего) установленного бита — то есть первого справа бита, равного 1 (нумерация обычно с нуля).

**Примеры:**  
- `n = 18` (`10010₂`) → младший установленный бит на индексе 1  
- `n = 12` (`1100₂`) → индекс 2  
- `n = 0b100000` → индекс 5

---

## Идея и подход

### 1. Получить маску самого младшего установленного бита

Изолировать этот бит можно так:

```python
rightmost = n & -n
```

- `-n` — это дополнительный код числа (инверсия всех битов + 1).
- В результате только самый правый установленный бит будет равен 1, остальные — 0.

### 2. Найти индекс этого бита

Чтобы узнать позицию (индекс) этого бита:

#### a) Через цикл

```python
def rightmost_set_bit_index(n):
    if n == 0:
        return -1  # Нет установленных битов
    pos = 0
    while (n & 1) == 0:
        n >>= 1
        pos += 1
    return pos

# Примеры:
print(rightmost_set_bit_index(18))  # 1
print(rightmost_set_bit_index(12))  # 2
print(rightmost_set_bit_index(32))  # 5
```

#### b) Через `bit_length`

Альтернативно, можно использовать `bit_length` и маску:

```python
def rightmost_set_bit_index(n):
    if n == 0:
        return -1
    mask = n & -n
    return mask.bit_length() - 1
```

---

## Как это работает

Для `n = 18` (`10010₂`):

- `-n`: дополнительный код для 18 — `01110₂` (в 5 битах)
- `n & -n = 10010₂ & 01110₂ = 00010₂` (то есть 2), бит в позиции 1

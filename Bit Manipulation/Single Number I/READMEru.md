# Single Number I (Уникальное число)

## Описание задачи

Дан не пустой массив целых чисел, в котором каждое число встречается ровно два раза, кроме одного. Найдите это единственное число.

---

## Подход к решению

Оптимальный способ — использовать операцию XOR:

- `a ^ a = 0`
- `a ^ 0 = a`
- XOR коммутативен и ассоциативен

Если сделать XOR всех элементов, парные числа уничтожатся, а останется только уникальное.

---

## Пример на Python

```python
def singleNumber(nums):
    result = 0
    for num in nums:
        result ^= num
    return result

# Пример:
nums = [4, 1, 2, 1, 2]
print(singleNumber(nums))  # 4
```

---

## Как это работает

- XOR одинаковых чисел даёт 0.
- XOR с нулём не меняет число.
- Все дубликаты обнуляются, остаётся только уникальный элемент.

---

## Применение

- Поиск уникального значения среди парных дубликатов.
- Задачи на битовые операции и оптимизацию памяти.
- Быстрая проверка целостности данных.

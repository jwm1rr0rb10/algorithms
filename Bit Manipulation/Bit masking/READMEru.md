# Bit Masking — Битовое Маскирование

## Описание Алгоритма

Битовое маскирование (Bit Masking) — это метод работы с отдельными битами числа с помощью битовых операций (AND, OR, XOR, NOT, сдвиги). Маска — это число, в котором определённые биты установлены в 1 или 0, чтобы выделить, изменить или проверить конкретные биты другого числа.

---

## Где применяется

- Хранение и обработка множества логических флагов в одном числе (например, разрешения доступа, состояния объектов).
- Оптимизация по памяти и скорости при работе с большими объёмами данных.
- Генерация всех подмножеств (power set) с помощью битовых представлений.
- Алгоритмы на графах, динамическое программирование по подмножествам.
- Игры, криптография, низкоуровневое программирование.

---

## Пример на Python

```python
x = 0b1010  # 10 в десятичной системе

# Установить 2-й бит (считая с 0)
x = x | (1 << 2)
print(bin(x))  # 0b1110

# Проверить, установлен ли 3-й бит
is_set = (x >> 3) & 1
print(is_set)  # 1 (бит установлен)

# Сбросить 1-й бит
x = x & ~(1 << 1)
print(bin(x))  # 0b1100
```

---

## Как это работает

1. **Установить бит:** `x | (1 << i)` — устанавливает i-й бит в 1.
2. **Сбросить бит:** `x & ~(1 << i)` — сбрасывает i-й бит в 0.
3. **Проверить бит:** `(x >> i) & 1` — возвращает значение i-го бита (0 или 1).
4. **Инвертировать бит:** `x ^ (1 << i)` — инвертирует i-й бит.

---

## Сложность

- **Время:** O(1) для основных операций (установить, сбросить, проверить бит).
- **Память:** O(1), если не хранить дополнительные данные.

---

## Полезные применения

- Представление подмножеств (например, для задач о рюкзаке).
- Сжатое хранение состояний, использование битовых масок вместо массивов булевых значений.
- Контроль разрешений и доступов (например, права пользователя).

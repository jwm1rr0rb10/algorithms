# Подмножества через биты

## Описание задачи

Дан массив (или множество) из `n` элементов. Требуется сгенерировать все возможные подмножества (степенное множество) с помощью битовых операций.

---

## Подход к решению

Каждое подмножество можно представить с помощью `n`-битного числа: каждый бит показывает, включён ли соответствующий элемент.

- Всего будет `2^n` подмножеств.
- Для каждого числа от `0` до `2^n - 1` двоичное представление определяет, какие элементы включены в подмножество.

---

## Пример на Python

```python
def subsets(nums):
    n = len(nums)
    result = []
    for mask in range(1 << n):  # от 0 до 2^n - 1
        subset = []
        for i in range(n):
            if (mask >> i) & 1:
                subset.append(nums[i])
        result.append(subset)
    return result

# Пример:
nums = [1, 2, 3]
print(subsets(nums))
# Вывод: [[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]
```

---

## Как это работает

- Перебираются все числа от `0` до `2^n - 1`.
- Биты каждого числа определяют, какие элементы брать в подмножество.

---

## Применение

- Генерация всех комбинаций/подмножеств для комбинаторных задач.
- Задачи типа Subset Sum, Knapsack и др.
- Полный перебор для задач на собеседованиях.

# Инверсия (переключение) i-го бита

## Описание задачи

Дано целое число `n` и позиция бита `i` (нумерация с 0 от младшего бита). Требуется инвертировать (переключить) i-й бит в числе `n`. Остальные биты не должны измениться.

---

## Подход к решению

- Используем побитовый XOR (`^`) с маской, где только i-й бит равен 1.
- Маска: `1 << i`.
- Операция `n ^ (1 << i)` меняет i-й бит (0 становится 1, 1 становится 0).

---

## Пример на Python

```python
def toggle_ith_bit(n, i):
    return n ^ (1 << i)

# Примеры:
print(bin(toggle_ith_bit(0b1010, 1)))  # 0b1010 ^ 0b0010 => 0b1000 (бит 1 был 1, стал 0)
print(bin(toggle_ith_bit(0b1010, 2)))  # 0b1010 ^ 0b0100 => 0b1110 (бит 2 был 0, стал 1)
print(bin(toggle_ith_bit(0b0, 3)))     # 0b0000 ^ 0b1000 => 0b1000 (бит 3 был 0, стал 1)
```

---

## Как это работает

- `1 << i` создаёт маску с единственным установленным битом.
- Побитовый XOR (`^`) инвертирует i-й бит: был 0 — станет 1, был 1 — станет 0.

---

## Применение

- Манипуляции с битовыми масками
- Переключение флагов, настроек, признаков
- Эффективное переключение состояний в играх и системном программировании

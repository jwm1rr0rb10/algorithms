# Поиск пропущенного числа с помощью XOR

## Описание задачи

Дан массив из `n` различных чисел в диапазоне от `0` до `n`.  
Найдите одно пропущенное число.

**Пример:**  
Вход: `[3, 0, 1]`  
Выход: `2`  
Пояснение: В диапазоне 0, 1, 2, 3 не хватает числа 2.

---

## Идея и подход

- Применяем XOR ко всем числам от `0` до `n`.
- Применяем XOR ко всем элементам массива.
- В результате останется только пропущенное число, так как все остальные "сотрутся" (x ^ x = 0).

---

## Пример на Python

```python
def find_missing_number(nums):
    n = len(nums)
    missing = 0
    for i in range(n + 1):
        missing ^= i
    for num in nums:
        missing ^= num
    return missing

# Примеры:
print(find_missing_number([3, 0, 1]))    # 2
print(find_missing_number([0, 1]))       # 2
print(find_missing_number([9,6,4,2,3,5,7,0,1])) # 8
```

---

## Как это работает

Для массива `[3, 0, 1]`:
- XOR всех чисел от 0 до 3: `0 ^ 1 ^ 2 ^ 3`
- XOR всех элементов массива: `3 ^ 0 ^ 1`
- Ответ: `(0 ^ 1 ^ 2 ^ 3) ^ (3 ^ 0 ^ 1) = 2`

## Преимущества

- Время: O(n)
- Память: O(1)
- Не требует дополнительных массивов или сортировки

---

## Применение

- Поиск пропущенных элементов в последовательностях
- Проверка целостности данных
- Выявление ошибок при передаче данных

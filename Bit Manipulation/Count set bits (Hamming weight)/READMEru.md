# Подсчёт количества единичных битов (Hamming weight)

## Описание задачи

Дано неотрицательное целое число `n`. Нужно посчитать количество битов, равных 1, в его двоичном представлении.  
Этот показатель также называют **вес Хэмминга** (Hamming weight).

**Пример:**  
- Ввод: `n = 13` (двоичное: `1101`)  
- Вывод: `3` (три единицы)

---

## Идея и подходы

Есть несколько способов посчитать количество единичных битов в числе:

### 1. Перебор каждого бита

- Пока `n` не равно нулю:
  - К счётчику прибавляем `n & 1` (проверяем младший бит).
  - Сдвигаем `n` вправо на 1 бит.

```python
def count_set_bits(n):
    count = 0
    while n:
        count += n & 1
        n >>= 1
    return count
```

### 2. Алгоритм Брайана Кернигана (эффективно для редких единиц)

- Пока `n` не равно нулю:
  - `n = n & (n - 1)` (обнуляет самый младший установленный бит).
  - Увеличиваем счётчик на 1.

```python
def count_set_bits(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
```

### 3. Встроенные функции Python

- Для Python 3.10 и выше:  
  `n.bit_count()`
- Для ранних версий:  
  `bin(n).count('1')`

## Применение

- Проверка количества установленных флагов в битовой маске.
- Подсчёт населения ("population count") в криптографии и кодах с исправлением ошибок.
- Оптимизация и битовые алгоритмы.

---

## Сложность

- **Время:** O(число битов) для перебора, O(число единичных битов) для метода Кернигана.
- **Память:** O(1)

# Обнуление i-го бита (Clear ith Bit)

## Описание задачи

Дано целое число `n` и позиция бита `i` (отсчёт с нуля справа налево, младший бит — индекс 0).  
Необходимо обнулить (сделать равным 0) i-й бит числа, остальные биты оставить без изменений.

**Пример:**  
Вход: `n = 13` (двоичное: `1101`), `i = 2`  
Результат: `n = 9` (двоичное: `1001`)

---

## Идея и подход

1. **Создать маску**, в которой только i-й бит равен 0, а остальные — 1.
2. **Выполнить побитовое И (`&`)** между числом и маской.
3. Это обнулит именно i-й бит, все остальные сохранятся.

**Как создать маску:**
- `(1 << i)` — получаем число, где только i-й бит равен 1.
- `~(1 << i)` — инвертируем: все биты 1, кроме i-го (он 0).

---

## Пример на Python

```python
def clear_ith_bit(n, i):
    mask = ~(1 << i)
    return n & mask

# Пример использования:
n = 13      # 1101
i = 2
print(clear_ith_bit(n, i))  # 9 (1001)
```

---

## Как это работает

1. Число 13 в двоичном виде: `1101`
2. `i = 2` — то есть третий справа бит (отсчёт с нуля).
3. Маска:
   - `(1 << 2)` = `0b100` (4)
   - `~4` = `...11111011` (все биты 1, кроме третьего справа)
4. `1101 & 1011 = 1001` (9 в десятичной системе)

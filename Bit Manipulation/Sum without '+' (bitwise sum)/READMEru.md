# Сумма без знака '+' (битовая сумма)

## Описание задачи

Даны два целых числа `a` и `b`. Нужно вычислить их сумму, не используя операторы `+` или `-`, только с помощью побитовых операций.

---

## Подход к решению

Симулируем сложение с помощью битовых операций:

- **XOR (`^`)** складывает биты без учета переносов.
- **AND (`&`)** со сдвигом влево (`<< 1`) вычисляет переносы.
- Повторяем: сумма становится новым `a`, перенос — новым `b`.
- Продолжаем, пока перенос не станет равен нулю.

---

## Пример на Python

```python
def getSum(a, b):
    MAX = 0xFFFFFFFF
    MASK = 0x7FFFFFFF
    while b != 0:
        carry = (a & b) << 1
        a = (a ^ b) & MAX
        b = carry & MAX
    # Для отрицательных чисел возвращаем значение с учётом знака
    return a if a <= MASK else ~(a ^ MAX)

# Примеры:
print(getSum(2, 3))    # 5
print(getSum(-1, 1))   # 0
```

---

## Как это работает

- XOR складывает биты без переносов.
- AND и сдвиг определяют, где должны быть переносы.
- Цикл продолжается, пока перенос не исчезнет.
- Для отрицательных чисел используется маскирование, чтобы эмулировать 32-битные знаковые числа.

- Задачи на собеседованиях по битовым трюкам и ограничениям на операторы.

# Деление двух целых чисел (битовое деление)

## Описание задачи

Даны два целых числа — `dividend` (делимое) и `divisor` (делитель).  
Вычислите их целочисленное частное (`dividend // divisor`), используя только побитовые операции (без `/`, `//`, `%` и `*`).

---

## Идея и подход

- Деление можно реализовать через повторное вычитание делителя из делимого, но для эффективности используется сдвиг делителя — чтобы за раз вычитать как можно больше.
- Получается аналог деления “в столбик”, но в двоичном виде.
- Знак результата отрицательный, если один из аргументов отрицательный, а другой положительный.

---

## Пример на Python

```python
def divide(dividend, divisor):
    if divisor == 0:
        raise ZeroDivisionError("division by zero")
    negative = (dividend < 0) != (divisor < 0)
    a, b = abs(dividend), abs(divisor)
    result = 0
    for i in range(31, -1, -1):
        if (a >> i) >= b:
            result += 1 << i
            a -= b << i
    return -result if negative else result

# Примеры:
print(divide(13, 3))    # 4
print(divide(-15, 4))   # -3
print(divide(20, -5))   # -4
print(divide(-100, -7)) # 14
```

---

## Как это работает

1. Берём модули обоих чисел.
2. Для каждой позиции бита (от старших к младшим):
    - Если `divisor << i` можно вычесть из текущего делимого, делаем это и добавляем `1 << i` к результату.
3. К результату применяем нужный знак (если только один из аргументов отрицателен — ответ отрицательный).

## Особенности

- Деление на ноль не допускается (`ZeroDivisionError`).
- В задачах с 32-битными числами результат обычно ограничивают диапазоном [-2^31, 2^31-1].
- Метод работает как для положительных, так и для отрицательных чисел.

## Применение

- В низкоуровневых библиотеках, микроконтроллерах и встраиваемых системах, где операция деления недоступна или медленная.
- В интервью и олимпиадных задачах для проверки понимания битовых операций.

# Проверка степени двойки

## Описание задачи

Дано целое число `n`. Нужно определить, является ли оно степенью двойки (то есть, можно ли записать как 2^k для целого k ≥ 0).

---

## Подход

Число — степень двойки, если оно положительно и в его двоичном представлении ровно один бит равен 1.

**Битовый трюк:**  
Для любого положительного числа n:  
если `(n & (n - 1)) == 0`, то n — степень двойки.

**Почему это работает:**  
- В двоичном виде степень двойки выглядит как `100...0`.
- Вычитая 1, все биты после единственного установленного бита инвертируются.
- Поэтому `n & (n - 1)` будет равно 0 только для степеней двойки.

---

## Пример на Python

```python
def isPowerOfTwo(n):
    return n > 0 and (n & (n - 1)) == 0

# Примеры:
print(isPowerOfTwo(1))    # True (2^0)
print(isPowerOfTwo(2))    # True (2^1)
print(isPowerOfTwo(3))    # False
print(isPowerOfTwo(8))    # True (2^3)
print(isPowerOfTwo(0))    # False
print(isPowerOfTwo(-8))   # False
```

---

## Как это работает

- Проверяем, что число положительное.
- Используем битовую операцию для проверки единственного установленного бита.

---

## Применение

- Быстрая проверка кратности размерам памяти/выравнивания
- Оптимизация алгоритмов, связанных с размерностями или контейнерами
- Битовые маски, графика, системное программирование

# Генерация подмножеств с помощью битовой маски

---

## Описание задачи

Дан массив из `n` элементов. Требуется сгенерировать все возможные подмножества (power set, множество всех подмножеств).

---

## Подход

Оптимальный способ — использование битовых масок:

1. **Каждое подмножество — это битовая маска:**  
   Для каждого числа от `0` до `2^n-1` двоичное представление — это маска: если `i`-й бит равен 1, то `i`-й элемент включён в подмножество.

2. **Перебор всех масок:**  
   Для каждой маски формируем подмножество.

---

## Пример на Python

```python
def subsets(nums):
    n = len(nums)
    all_subsets = []
    for mask in range(1 << n):  # 0..2^n-1
        subset = []
        for i in range(n):
            if mask & (1 << i):
                subset.append(nums[i])
        all_subsets.append(subset)
    return all_subsets

# Пример:
nums = [1, 2, 3]
print(subsets(nums))
# Вывод:
# [[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]
```

---

## Как это работает

- Всего существует `2^n` возможных масок.
- Для каждой маски перебираем биты: если бит установлен, включаем соответствующий элемент в подмножество.

---

## Применение

- Перебор всех комбинаций (подмножеств) множества
- Bitmask DP (динамическое программирование по маске)
- Задачи на сумму подмножеств, разбиение, комбинаторику

---

## Когда использовать

- Если `n` небольшой (≤ 20–22)
- Для задач, где требуется перебрать все подмножества

---

## Когда не стоит использовать

- Если `n` большой: экспоненциальная сложность

---

## Сложность

- **Время:** O(n × 2^n)  
  (для каждого из 2^n подмножеств выполняется до n проверок)
- **Память:** O(n × 2^n) (если храним все подмножества)

# Максимальное произведение длин слов без общих букв

## Описание задачи

Дан список слов. Нужно найти две такие строки, которые не имеют общих букв, и максимизировать произведение их длин. Если таких пар нет — вернуть 0.

---

## Подход к решению

1. **Битовые маски:**  
   Для каждого слова строим 26-битную маску по латинским буквам (a-z). Если буква есть в слове — соответствующий бит равен 1.
2. **Сравнение пар:**  
   Перебираем все пары слов и проверяем, нет ли общих букв (`mask1 & mask2 == 0`). Если нет — считаем произведение длин и обновляем максимум.

---

## Пример на Python

```python
def maxProduct(words):
    n = len(words)
    masks = [0] * n
    lengths = [len(word) for word in words]
    
    for i, word in enumerate(words):
        for c in word:
            masks[i] |= 1 << (ord(c) - ord('a'))
    
    max_prod = 0
    for i in range(n):
        for j in range(i + 1, n):
            if masks[i] & masks[j] == 0:
                max_prod = max(max_prod, lengths[i] * lengths[j])
    return max_prod

# Пример использования:
words = ["abcw", "baz", "foo", "bar", "xtfn", "abcdef"]
print(maxProduct(words))  # 16 (слова "abcw" и "xtfn")
```

---

## Как это работает

- **Битовые маски** позволяют быстро проверить наличие общих букв между словами.
- **Перебор пар** даёт максимум среди всех подходящих пар.

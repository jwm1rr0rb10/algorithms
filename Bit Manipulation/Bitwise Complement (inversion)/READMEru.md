# Побитовая инверсия (Bitwise Complement)

## Описание задачи

Дано целое число `n`. Необходимо найти его побитовую инверсию.  
Побитовая инверсия числа — это число, полученное заменой всех его битов на противоположные: 0 становится 1, 1 становится 0.  
Результат должен учитывать только **значащие биты** исходного числа (игнорировать ведущие нули).

**Пример:**  
Ввод: `n = 5`  
Двоичное представление: `101`  
Побитовая инверсия: `010`  
Вывод: `2`

---

## Идея и подход

- Побитовая инверсия меняет каждый бит числа на противоположный.
- Для заданного числа инвертируются только значащие биты (до старшей единицы).
- Для этого создают маску с тем же количеством бит, что и у исходного числа (все биты маски равны 1).
- Инверсия считается как `n ^ mask` (XOR с маской).

---

## Пример на Python

```python
def bitwiseComplement(n):
    if n == 0:
        return 1
    mask = (1 << n.bit_length()) - 1
    return n ^ mask

# Пример использования:
print(bitwiseComplement(5))  # 2
```

---

## Как это работает

1. Если `n` равно нулю, его инверсия — 1 (один бит).
2. Определяется длина двоичного представления числа (без ведущих нулей).
3. Формируется маска из единиц такой же длины.
4. Инверсия вычисляется как XOR исходного числа и маски.

# Побитовая инверсия — пример на Python

Дано целое число `n`. Найдите его побитовую инверсию (инвертировать все значащие биты до старшей единицы).

## Реализация на Python

```python
def bitwiseComplement(n):
    if n == 0:
        return 1
    mask = (1 << n.bit_length()) - 1
    return n ^ mask

# Примеры использования:
print(bitwiseComplement(5))   # Вывод: 2
print(bitwiseComplement(0))   # Вывод: 1
print(bitwiseComplement(8))   # Вывод: 7
```

## Пошаговое объяснение

1. **Отдельный случай для нуля:**  
   Инверсия 0 — это 1 (один бит).

2. **Определяем количество значащих бит:**  
   `n.bit_length()` — сколько бит нужно для записи числа `n`.

3. **Создаём маску:**  
   `(1 << n.bit_length()) - 1` — маска, где все значащие биты равны 1.  
   Например, для `n = 5 (101)` маска будет `111` (7).

4. **XOR с маской:**  
   `n ^ mask` инвертирует все значащие биты.  
   Для `n = 5 (101)` и маски `111` результат — `010` (2).

## Пример: инверсия ASCII-кода в Python

```python
c = 'A'
ascii_code = ord(c)
print("Исходное:", bin(ascii_code))
mask = (1 << ascii_code.bit_length()) - 1
print("Инверсия:", bin(ascii_code ^ mask))
```

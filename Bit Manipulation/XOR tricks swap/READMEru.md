# XOR-трюк для обмена значениями (XOR swap)

## Описание задачи

Поменять местами значения двух переменных `a` и `b` без использования временной переменной, только с помощью побитовых операций.

---

## Подход к решению

Классический XOR swap алгоритм:

1. `a = a ^ b`
2. `b = a ^ b`  (теперь в `b` лежит исходное значение `a`)
3. `a = a ^ b`  (теперь в `a` лежит исходное значение `b`)

Временные переменные не нужны.

---

## Пример на Python

```python
a = 5
b = 9

# XOR swap:
a = a ^ b
b = a ^ b
a = a ^ b

print(a, b)  # Вывод: 9 5
```

---

## Как это работает

- После `a = a ^ b` в `a` лежит XOR двух значений.
- `b = a ^ b` извлекает исходное значение `a` в `b`.
- `a = a ^ b` извлекает исходное значение `b` в `a`.

---

## Важные замечания

- Если `a` и `b` — это одна и та же переменная (один и тот же адрес), результат будет нулём.
- В современных языках программирования этот трюк — скорее академический или для собеседований; обмен с временной переменной обычно понятнее и не медленнее.

---

## Применение

- Низкоуровневое программирование, микроконтроллеры, задачки на собеседованиях.
- Экстремальная экономия памяти (редко встречается).

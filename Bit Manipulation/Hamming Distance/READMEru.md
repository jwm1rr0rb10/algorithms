# Расстояние Хэмминга

## Описание задачи

**Расстояние Хэмминга** между двумя числами — это количество позиций, в которых их двоичные представления различаются.

**Пример:**  
- `x = 3` (`011` в двоичной)  
- `y = 5` (`101` в двоичной)  
- Отличия в позициях 0 и 2 ⇒ расстояние Хэмминга = 2

---

## Идея и подход

1. Найти XOR чисел: `diff = x ^ y`  
   В результате — 1 в тех разрядах, где биты различаются.
2. Посчитать количество единичных битов в diff — это и есть расстояние Хэмминга.

---

## Пример на Python

```python
def hamming_distance(x, y):
    diff = x ^ y
    count = 0
    while diff:
        count += diff & 1
        diff >>= 1
    return count

# Или через встроенные функции:
def hamming_distance_fast(x, y):
    return bin(x ^ y).count('1')

# Примеры:
print(hamming_distance(3, 5))      # 2
print(hamming_distance(1, 4))      # 2 (001 vs 100)
print(hamming_distance(7, 15))     # 1 (0111 vs 1111: разница только в 3-м бите)
```

---

## Как это работает

- XOR ставит 1 там, где биты различаются.
- Подсчёт единиц — это и есть ответ.

## Применение

- Коды с обнаружением и исправлением ошибок (ECC, паритет)
- Сравнение битовых строк, масок признаков
- Генетические алгоритмы, машинное обучение

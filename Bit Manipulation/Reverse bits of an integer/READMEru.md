# Реверс битов у целого числа

## Описание задачи

Дано 32-битное беззнаковое целое число. Необходимо развернуть порядок битов в числе и вернуть результат.

**Пример:**  
Вход: `43261596` (`00000010100101000001111010011100` в двоичном виде)  
Выход: `964176192` (`00111001011110000010100101000000` в двоичном виде)

---

## Подход к решению

1. **Проходим по всем 32 битам** числа.
2. **Сдвигаем результат влево на 1** и добавляем младший бит исходного числа.
3. **Сдвигаем исходное число вправо на 1**.
4. Повторяем 32 раза.

---

## Пример на Python

```python
def reverseBits(n):
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result

# Пример использования:
x = 43261596
print(reverseBits(x))  # 964176192
```

---

## Как это работает

- Каждый раз берём младший бит `n` и добавляем его к результату слева.
- За 32 итерации все биты окажутся в обратном порядке.

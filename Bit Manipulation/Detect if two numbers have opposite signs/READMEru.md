# Определение противоположных знаков у двух чисел

## Описание задачи

Даны два целых числа `a` и `b`. Нужно определить, имеют ли они противоположные знаки (то есть одно положительное, а другое отрицательное).

---

## Идея и подход

- В двоичном представлении знак числа определяется старшим (самым левым) битом — знаковым битом.
- Если два числа имеют разные знаки, их знаковые биты будут различаться.
- Побитовое исключающее ИЛИ (`^`) для чисел с разными знаками даст результат, у которого знаковый бит будет 1 (то есть отрицательное число).
- Поэтому, если `(a ^ b) < 0`, значит, у чисел противоположные знаки.

---

## Пример на Python

```python
def have_opposite_signs(a, b):
    return (a ^ b) < 0

# Примеры:
print(have_opposite_signs(5, -3))    # True
print(have_opposite_signs(-7, 12))   # True
print(have_opposite_signs(8, 19))    # False
print(have_opposite_signs(-4, -100)) # False
```

---

## Как это работает

- Оператор `^` (XOR) сравнивает каждый бит чисел; для знакового бита, если знаки разные, результат будет отрицательным.
- Проверка `a ^ b < 0` позволяет быстро узнать, были ли исходные числа с разными знаками.

## Применение

- Полезно для низкоуровневого программирования, алгоритмов и оптимизаций.
- Быстрая проверка разности знаков без дополнительных условий.
- Может использоваться для вычисления модуля, обнаружения переполнения и др.

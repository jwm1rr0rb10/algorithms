# Karatsuba Algorithm — Разделяй и властвуй (Divide & Conquer)

## Описание задачи

**Алгоритм Карацубы** — это быстрый способ перемножения двух больших чисел, использующий принцип "разделяй и властвуй". Позволяет перемножать числа быстрее, чем традиционный "школьный" метод.

**Задача:**  
Даны два больших числа (как строки или массивы цифр). Требуется вычислить их произведение.

**Пример:**
```python
x = 1234
y = 5678
# Ответ: 7006652
```

## Алгоритм (Karatsuba)

1. Разделить каждое число на две части (старшие и младшие разряды):
   - x = x1 * 10^m + x0
   - y = y1 * 10^m + y0
2. Рекурсивно вычислить:
   - z0 = x0 * y0
   - z1 = (x0 + x1) * (y0 + y1)
   - z2 = x1 * y1
3. Собрать результат:
   - x * y = z2 * 10^(2m) + (z1 - z2 - z0) * 10^m + z0

### Пример кода на Python

```python
def karatsuba(x, y):
    # Базовый случай для маленьких чисел
    if x < 10 or y < 10:
        return x * y
    n = max(len(str(x)), len(str(y)))
    m = n // 2

    # Разделение чисел
    x1, x0 = divmod(x, 10**m)
    y1, y0 = divmod(y, 10**m)

    # Рекурсивные вычисления
    z0 = karatsuba(x0, y0)
    z2 = karatsuba(x1, y1)
    z1 = karatsuba(x0 + x1, y0 + y1) - z2 - z0

    return z2 * 10**(2*m) + z1 * 10**m + z0

# Пример использования:
x = 1234
y = 5678
print(karatsuba(x, y))  # 7006652
```

## Временная сложность

- O(n^log2(3)) ≈ O(n^1.585), где n — количество разрядов числа.

## Где применять

- При работе с длинной арифметикой (большие числа — криптография, большие вычисления).
- В реализации калькуляторов, символьных вычислений, научных библиотек.

## Недостатки

- Для очень маленьких чисел проще и быстрее обычное умножение.
- Требует дополнительной памяти для рекурсии.

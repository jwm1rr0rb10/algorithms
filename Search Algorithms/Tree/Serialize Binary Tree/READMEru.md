# Сериализация бинарного дерева (Serialize Binary Tree)

---

## Описание задачи

**Сериализация бинарного дерева** — это преобразование структуры бинарного дерева в строку или массив, чтобы её можно было сохранить или передать (например, по сети).  
Чаще всего используют формат ширины (BFS, level order), подставляя `null` для отсутствующих детей.

---

## Подход (алгоритм)

1. Используем обход в ширину (BFS, с очередью).
2. Для каждого посещённого узла:
   - Если узел не `None` — сохраняем его значение и добавляем детей в очередь.
   - Если узел `None` — сохраняем `null` в результат.
3. В конце можно удалить лишние `null` с конца массива (по желанию).

---

## Пример на Python

```python
from collections import deque

class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

def serialize(root):
    if not root:
        return "[]"
    queue = deque([root])
    res = []
    while queue:
        node = queue.popleft()
        if node:
            res.append(str(node.val))
            queue.append(node.left)
            queue.append(node.right)
        else:
            res.append("null")
    # Удаляем лишние null в конце (опционально)
    while res and res[-1] == "null":
        res.pop()
    return "[" + ",".join(res) + "]"

# Пример использования:
# Пусть tree — корень бинарного дерева
# print(serialize(tree))  # Например, "[1,2,3,null,null,4,5]"
```

---

## Применение

- Сохранение и передача структуры дерева между программами, по сети, в тестах.
- Часто используется вместе с десериализацией для проверки корректности алгоритмов на деревьях.

---

## Когда использовать

- Когда нужно сохранить/передать дерево в виде строки или массива.
- Для задач LeetCode, интервью, олимпиад.

---

## Когда не стоит использовать

- Для очень больших деревьев (может быть избыточно по памяти).
- Если нужно сериализовать только значения (без структуры).

---

## Сложность

- **Время:** O(n), где n — количество узлов
- **Память:** O(n)

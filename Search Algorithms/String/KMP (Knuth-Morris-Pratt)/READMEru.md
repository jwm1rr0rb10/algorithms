# Алгоритм Кнута-Морриса-Пратта (KMP)

## Где применяется?

**KMP (Кнут-Моррис-Пратт)** — это эффективный алгоритм поиска подстроки, который широко используется для:
- Поиска всех вхождений шаблона в тексте.
- Обработки строк в редакторах, компиляторах, задачах биоинформатики (например, поиск по ДНК).
- Любых задач, где требуется быстрое и гарантированно эффективное сканирование текста на наличие подстроки.

---

## Временная и пространственная сложность

- **Время:** O(n + m), где n — длина текста, m — длина шаблона.
    - O(m) на построение префикс-функции.
    - O(n) на поиск.
- **Память:** O(m) — для хранения префикс-функции.

---

## Как работает

1. **Построение префикс-функции** (failure function) для шаблона:
    - Для каждой позиции в шаблоне вычисляется длина наибольшего собственного префикса, который совпадает с суффиксом подстроки, заканчивающейся в этой позиции.
2. **Поиск шаблона в тексте**:
    - Используя префикс-функцию, при несовпадении символов не возвращаемся к началу шаблона, а "перепрыгиваем" к следующей потенциальной позиции.
    - Это позволяет избежать повторных проверок заведомо неподходящих символов.

---

## Пример на Python с пояснением

```python
def compute_prefix_function(pattern):
    m = len(pattern)
    lps = [0] * m
    length = 0

    for i in range(1, m):
        while length > 0 and pattern[i] != pattern[length]:
            length = lps[length - 1]
        if pattern[i] == pattern[length]:
            length += 1
            lps[i] = length
    return lps

def kmp_search(text, pattern):
    n, m = len(text), len(pattern)
    if m == 0:
        return []
    lps = compute_prefix_function(pattern)
    result = []
    i = j = 0

    while i < n:
        if pattern[j] == text[i]:
            i += 1
            j += 1
            if j == m:
                result.append(i - m)
                j = lps[j - 1]
        else:
            if j != 0:
                j = lps[j - 1]
            else:
                i += 1
    return result

# Пример использования:
text = "ababcababcabc"
pattern = "ababc"
print(kmp_search(text, pattern))  # Выведет: [0, 5]
```

### Пояснения

- Сначала строится массив lps (longest proper prefix which is also suffix, он же "префикс-функция").
- Основной цикл сравнивает символы текста и шаблона. Если символы совпадают, продвигаем оба индекса.
- Если достигли конца шаблона (j == m), найдено вхождение — добавляем индекс в результат.
- При несовпадении используем lps, чтобы не возвращаться к началу шаблона, а перескочить к позиции с потенциальным совпадением.
- Функция возвращает список индексов, где найден шаблон.

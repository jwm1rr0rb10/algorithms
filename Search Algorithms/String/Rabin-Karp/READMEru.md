# Алгоритм Рабина-Карпа

## Где применяется?

**Рабин-Карп** — это алгоритм поиска подстроки, который широко используется для:
- Поиска всех вхождений шаблона в тексте.
- Проверки на плагиат (поиск совпадающих фрагментов в больших документах).
- Поиска сразу нескольких шаблонов (с модификациями алгоритма).
- Быстрого поиска в ДНК-последовательностях и других задачах биоинформатики, где требуется сопоставление подстрок.

---

## Временная и пространственная сложность

- **Время (средний/ожидаемый случай):** O(n + m), где n — длина текста, m — длина шаблона.
    - Алгоритм использует хеширование, поэтому сравнение хешей выполняется быстро.
    - В худшем случае (если хеши совпадают, а строки разные) — до O(n·m).
- **Память:** O(1) (если не учитывать хранение самого текста и шаблона).
    - Иногда считают O(m) для хранения шаблона и O(1) для текущих хешей.

---

## Как работает

1. Считается хеш шаблона.
2. Считается хеш первого окна такой же длины в тексте.
3. Сравниваются хеши — если совпадают, производится посимвольное сравнение строк.
4. Окно сдвигается на один символ, хеш обновляется с помощью "скользящего" хеша (rolling hash).
5. Повторяется до конца текста.

---

## Пример на Python с пояснением

```python
def rabin_karp(text, pattern):
    n, m = len(text), len(pattern)
    if m > n:
        return []

    base = 256  # количество возможных символов (например, для ASCII)
    mod = 10**9 + 7  # большое простое число для предотвращения переполнения

    pattern_hash = 0
    window_hash = 0
    h = 1  # base^(m-1) % mod

    for i in range(m - 1):
        h = (h * base) % mod

    for i in range(m):
        pattern_hash = (base * pattern_hash + ord(pattern[i])) % mod
        window_hash = (base * window_hash + ord(text[i])) % mod

    result = []
    for i in range(n - m + 1):
        # Если хеши совпали, сверяем строки посимвольно
        if pattern_hash == window_hash:
            if text[i:i + m] == pattern:
                result.append(i)
        # Обновляем хеш для следующего окна
        if i < n - m:
            window_hash = (base * (window_hash - ord(text[i]) * h) + ord(text[i + m])) % mod
            if window_hash < 0:
                window_hash += mod
    return result

# Пример использования:
text = "abracadabra"
pattern = "abra"
print(rabin_karp(text, pattern))  # Выведет: [0, 7]
```

### Пояснения

- `base` — основание системы счисления (например, 256 для ASCII-символов).
- `mod` — большое простое число для уменьшения вероятности коллизий и избежания переполнения.
- `h` — коэффициент для эффективного удаления первого символа из скользящего окна.
- Сначала вычисляются хеши шаблона и первого окна.
- Для каждого окна:
    - Если хеши совпали, производится посимвольная проверка.
    - Rolling hash позволяет быстро считать хеш для следующего окна.
- Функция возвращает список индексов, где найден шаблон.

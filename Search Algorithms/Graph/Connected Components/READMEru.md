# Компоненты связности в неориентированном графе

---

## Описание задачи

Дан неориентированный граф. Найти все его компоненты связности.  
Компонента связности — это группа вершин, между которыми существует путь, и которые не соединены с остальными вершинами графа.

---

## Подход

Используем обход в глубину (DFS) или в ширину (BFS):

1. Для каждой непосещённой вершины запускаем обход (DFS/BFS).
2. Все достижимые из неё вершины отмечаем как посещённые — они образуют одну компоненту.
3. Повторяем для всех вершин.

---

## Пример на Python

```python
def connected_components(graph):
    visited = set()
    components = []

    def dfs(node, comp):
        visited.add(node)
        comp.add(node)
        for neighbor in graph[node]:
            if neighbor not in visited:
                dfs(neighbor, comp)

    for node in graph:
        if node not in visited:
            comp = set()
            dfs(node, comp)
            components.append(comp)
    return components

# Пример:
graph = {
    0: [1, 2],
    1: [0],
    2: [0],
    3: [4],
    4: [3]
}
print(connected_components(graph))
# Вывод: [{0, 1, 2}, {3, 4}]
```

---

## Как это работает

- Каждый запуск обхода находит “островок” достижимых вершин.
- Все посещённые вершины отмечаются, чтобы не пересчитывать их повторно.

---

## Применение

- Поиск изолированных кластеров в сетях (социальных, дорожных, коммуникационных)
- Предобработка для других алгоритмов на графах
- Анализ связности

---

## Когда использовать

- Когда нужно найти все отдельные “островки” (кластеры) в графе.

---

## Когда не стоит использовать

- Для ориентированных графов: нужен поиск сильно связных компонент (SCC).
- Если нужно только проверить, связен ли граф.

---

## Сложность

- **Время:** O(V + E)
- **Память:** O(V)

# Алгоритм Краскала (Kruskal’s Algorithm)

## Что это такое?

**Алгоритм Краскала** — классический жадный алгоритм для поиска минимального остовного дерева (MST) в неориентированном взвешенном графе.

- **Минимальное остовное дерево** — это подмножество рёбер, соединяющее все вершины с минимальной возможной суммой весов и не образующее циклов.

---

## Как работает?

1. **Сортируем все рёбра графа** по неубыванию веса.
2. **Инициализируем остовное дерево как пустое.**
3. **Проходим по отсортированным рёбрам:**
   - Для каждого ребра проверяем: соединяет ли оно две разные компоненты (т.е. не создаёт цикл)?
   - Если да — добавляем ребро в остовное дерево.
   - Если образуется цикл, ребро пропускаем.
4. **Останавливаемся, когда в остовном дереве (V-1) рёбер** (V — число вершин).

Для проверки циклов используют структуру **система непересекающихся множеств** (DSU, Union-Find).

---

## Где применяется?

- Строительство сетей (электричество, дороги, компьютерные сети) с минимальными затратами.
- Кластеризация в анализе данных.
- Оптимизация соединений и минимизация стоимости.

---

## Особенности

- Работает для неориентированных графов (в том числе с отрицательными весами рёбер).
- Простой и эффективный для больших графов.

---

## Сложность

| Сложность        | Значение    |
|:----------------:|:----------:|
| **Время**        | O(E log E) |
| **Память**       | O(V)       |

- **E** — число рёбер  
- **V** — число вершин

---

## Пример на Python

```python
class DSU:
    def __init__(self, n):
        self.parent = list(range(n))

    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]

    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)
        if xr == yr:
            return False
        self.parent[yr] = xr
        return True

def kruskal(n, edges):
    """
    n: количество вершин (вершины от 0 до n-1)
    edges: список (вес, u, v)
    Возвращает: список рёбер в MST и суммарный вес
    """
    edges.sort()
    dsu = DSU(n)
    mst = []
    total_weight = 0

    for w, u, v in edges:
        if dsu.union(u, v):
            mst.append((u, v, w))
            total_weight += w
            if len(mst) == n - 1:
                break

    return mst, total_weight

# Пример использования
edges = [
    (1, 0, 1),
    (3, 0, 2),
    (2, 1, 2),
    (4, 1, 3),
    (5, 2, 3),
]
n = 4
mst, total = kruskal(n, edges)
print("Рёбра в остовном дереве:", mst)
print("Суммарный вес:", total)
```

# Алгоритм Прима (Prim's Algorithm)

Алгоритм Прима — это жадный алгоритм для поиска минимального остовного дерева (MST, minimum spanning tree) в связном взвешенном графе. Минимальное остовное дерево — это подмножество рёбер, соединяющее все вершины графа с минимально возможной суммой весов и без циклов.

## Краткое описание

1. Выбрать любую вершину в качестве начальной.
2. Добавлять к дереву самое дешёвое ребро, соединяющее уже построенное дерево с одной из оставшихся вершин (не допускается образование цикла).
3. Повторять шаг 2, пока в дереве не окажутся все вершины графа.

## Применение

- Прокладка кабелей между зданиями с минимальными затратами.
- Построение дорожных сетей.
- Сетевые инфраструктуры и т.п.

## Пример на Python

```python
import heapq

def prim(graph, start=0):
    n = len(graph)
    visited = [False] * n
    min_edges = [(0, start)]  # (вес, вершина)
    total_weight = 0

    while min_edges:
        weight, u = heapq.heappop(min_edges)
        if visited[u]:
            continue
        visited[u] = True
        total_weight += weight

        for v, w in graph[u]:
            if not visited[v]:
                heapq.heappush(min_edges, (w, v))

    return total_weight

# Пример графа: список смежности (индексы вершин, веса рёбер)
# Вершины: 0, 1, 2, 3
# Рёбра: (0-1, 1), (0-2, 4), (1-2, 2), (1-3, 6), (2-3, 3)
graph = [
    [(1,1), (2,4)],      # 0
    [(0,1), (2,2), (3,6)], # 1
    [(0,4), (1,2), (3,3)], # 2
    [(1,6), (2,3)]         # 3
]

print("Суммарный вес MST:", prim(graph))  # Вывод: 6
```

## Визуализация

```
   1
0-----1
|\   /|
| 4 2 6
|/     \
2-------3
    3
```
Минимальное остовное дерево: рёбра (0-1, 1), (1-2, 2), (2-3, 3). Суммарный вес: 6.

## Сложность

- Время: O(E log V), где E — количество рёбер, V — количество вершин (с использованием кучи).
- Память: O(V + E).

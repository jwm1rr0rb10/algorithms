# Поиск в бинарном дереве поиска (BST)

## Что такое бинарное дерево поиска (BST)?

**Бинарное дерево поиска (BST)** — это вид бинарного дерева, в котором для каждого узла выполняется правило:
- Все значения в левом поддереве **меньше** значения узла.
- Все значения в правом поддереве **больше** значения узла.
- Это правило рекурсивно для всех потомков.

Пример BST:
```
      8
     / \
    3   10
   / \    \
  1   6    14
     / \   /
    4   7 13
```

---

## Как работает поиск в BST

Поиск в BST использует свойство упорядоченности:

1. **Начинаем с корня.**
2. Если значение узла равно искомому — элемент найден.
3. Если искомое значение **меньше** значения узла — идём в левое поддерево.
4. Если искомое значение **больше** — идём в правое поддерево.
5. Повторяем шаги 2–4, пока не найдём элемент или не дойдём до пустого указателя (значение отсутствует).

---

## Сложность

- **Время:**
    - **Лучший/средний случай (сбалансированное дерево):** O(log n)
    - **Худший случай (дерево вырождено в список):** O(n)
- **Память:**
    - Итеративный поиск: O(1)
    - Рекурсивный поиск: O(h), где h — высота дерева

---

## Когда использовать

- Данные часто изменяются (добавление, удаление, поиск)
- Нужно поддерживать упорядоченное хранение и быстрые диапазонные запросы

## Когда не подходит

- Дерево несбалансировано (может деградировать в список)
- Если требуется только быстрый поиск, без изменений — проще использовать отсортированный массив и бинарный поиск

---

## Пример на Python

```python
class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def bst_search(root, target):
    current = root
    while current:
        if current.val == target:
            return current
        elif target < current.val:
            current = current.left
        else:
            current = current.right
    return None

# Пример использования
root = TreeNode(8)
root.left = TreeNode(3)
root.right = TreeNode(10)
root.left.left = TreeNode(1)
root.left.right = TreeNode(6)
root.left.right.left = TreeNode(4)
root.left.right.right = TreeNode(7)
root.right.right = TreeNode(14)
root.right.right.left = TreeNode(13)

result = bst_search(root, 6)
print("Найдено!" if result else "Не найдено.")
```

---

## Итог

Поиск в BST — это эффективный способ поиска элементов в динамически изменяемой, отсортированной структуре — **если дерево сбалансировано**. Для гарантированной эффективности стоит использовать самобалансирующиеся BST (например, AVL или красно-черные деревья).

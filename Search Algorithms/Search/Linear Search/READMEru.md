# Линейный поиск

**Что это такое:**  
Линейный поиск (или последовательный поиск) — самый простой алгоритм поиска элемента в массиве или списке. Он проверяет элементы по порядку, один за другим, пока не найдёт искомое значение или не дойдёт до конца.

---

**Как работает:**

1. Начинаем с первого элемента массива или списка.
2. Сравниваем текущий элемент с искомым значением.
3. Если элементы совпадают — возвращаем его индекс (или сам элемент).
4. Если нет — переходим к следующему элементу.
5. Повторяем шаги 2–4, пока не пройдём весь массив.
6. Если элемент не найден, возвращаем специальное значение (например, -1 или None).

---

## Сложность

- **Временная сложность:**  
  - Лучший случай: O(1) (элемент сразу первый)
  - Средний/худший случай: O(n) (нужно просмотреть все элементы)
- **По памяти:**  
  - O(1) (не нужны дополнительные структуры данных)

---

## Когда использовать

- Массив или список **не отсортированы**.
- Данных **мало**.
- Нужно найти **все вхождения** значения.

## Когда не использовать

- Данные **большие** (работает медленно).
- Данные **отсортированы** (бинарный поиск быстрее).
- Требуется **много частых поисков** (лучше использовать хеш-таблицы и другие структуры).

---

## Пример на Python

```python
def linear_search(data, target):
    """
    Возвращает индекс первого вхождения target в data,
    либо -1, если target не найден.
    """
    for index, value in enumerate(data):
        if value == target:
            return index
    return -1

# Пример использования:
numbers = [8, 3, 7, 1, 9, 4]
target1 = 7
target2 = 5

result1 = linear_search(numbers, target1)
print(f"Элемент {target1} найден по индексу {result1}" if result1 != -1 else f"Элемент {target1} не найден")

result2 = linear_search(numbers, target2)
print(f"Элемент {target2} найден по индексу {result2}" if result2 != -1 else f"Элемент {target2} не найден")
```

---

## Итог

Линейный поиск — универсальный и простой, но медленный для больших наборов данных. Используйте его для неотсортированных или небольших коллекций, либо при поиске всех совпадений.

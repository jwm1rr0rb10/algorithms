# Задача о рюкзаке (0/1 Knapsack) — Динамическое программирование

## Описание задачи

Дано `n` предметов, у каждого есть вес и ценность, и рюкзак вместимостью `W`.  
Нужно выбрать подмножество предметов (каждый можно взять не более одного раза), чтобы суммарная ценность была максимальной, а вес не превышал `W`.

**Пример:**  
Вход: weights = [1,3,4,5], values = [1,4,5,7], W = 7  
Выход: 9  
Пояснение: Берём предметы с весами 3 и 4 (ценности 4 + 5).

---

## Идея и подход

- Используем динамическое программирование:
  - Пусть `dp[i][w]` — максимальная ценность среди первых `i` предметов и веса не более `w`.
  - Для каждого предмета решаем: брать его (если хватает веса) или нет.
  - Переходы:
    - Если weights[i-1] > w: dp[i][w] = dp[i-1][w]
    - Иначе: dp[i][w] = max(dp[i-1][w], values[i-1] + dp[i-1][w - weights[i-1]])

---

## Пример на Python

```python
def knapsack(weights, values, W):
    n = len(weights)
    dp = [[0] * (W + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(W + 1):
            if weights[i-1] > w:
                dp[i][w] = dp[i-1][w]
            else:
                dp[i][w] = max(dp[i-1][w], values[i-1] + dp[i-1][w - weights[i-1]])
    return dp[n][W]

# Пример использования:
print(knapsack([1,3,4,5], [1,4,5,7], 7))  # Вывод: 9
```

---

## Анализ сложности

- **Время:** O(n * W)
- **Память:** O(n * W), можно оптимизировать до O(W) с помощью одномерного массива.

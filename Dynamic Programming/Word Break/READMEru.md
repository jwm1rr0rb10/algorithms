# Задача Word Break — Динамическое программирование

## Описание задачи

Дана строка `s` и словарь строк `wordDict`. Определить, можно ли разбить строку `s` на последовательность одного или более слов из словаря.

Варианты:
- Просто проверить, возможно ли такое разбиение (True/False).
- Иногда: вывести само разбиение.

---

## Идея алгоритма и подход

- Используется динамическое программирование:
  - Пусть `dp[i]` — можно ли разбить первые `i` символов строки `s` на слова из словаря.
  - Инициализация: `dp[0] = True` (пустая строка разбивается всегда).
  - Для каждой позиции `i` от 1 до `len(s)`:
    - Для каждого `j` от 0 до `i`:
      - Если `dp[j]` — True и `s[j:i]` есть в словаре, то `dp[i] = True`.
      - Прервать внутренний цикл, если `dp[i]` уже True.

---

## Пример на Python: Word Break

```python
def word_break(s, wordDict):
    word_set = set(wordDict)
    n = len(s)
    dp = [False] * (n + 1)
    dp[0] = True
    for i in range(1, n + 1):
        for j in range(i):
            if dp[j] and s[j:i] in word_set:
                dp[i] = True
                break
    return dp[n]

# Пример использования:
s = "leetcode"
wordDict = ["leet", "code"]
print(word_break(s, wordDict))  # Вывод: True
```

---

## Анализ сложности

- **Время:** O(n²)
- **Память:** O(n)
  - n — длина строки

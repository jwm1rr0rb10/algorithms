# Алгоритм мемоизации

## Что такое мемоизация?

**Мемоизация** — это метод оптимизации, который позволяет ускорять работу программ за счёт сохранения (кэширования) результатов дорогостоящих вычислений и возврата уже сохранённого результата при повторном обращении с теми же входными данными. Особенно полезен в рекурсивных алгоритмах с пересекающимися подзадачами, например, в динамическом программировании.

---

## Как работает мемоизация

1. **Проверка кэша:** Перед вычислением результата проверяем, нет ли его уже в кэше.
2. **Вычисление при необходимости:** Если нет — вычисляем результат и сохраняем его.
3. **Возврат результата:** Всегда возвращаем сохранённый результат для одинаковых входных данных.

---

## Пример: Числа Фибоначчи (Python)

Без мемоизации (медленно):

```python
def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)
```

С мемоизацией (быстро):

```python
def fib(n, memo={}):
    if n <= 1:
        return n
    if n not in memo:
        memo[n] = fib(n-1, memo) + fib(n-2, memo)
    return memo[n]
```

---

## Когда использовать мемоизацию

- Если ваш алгоритм решает одни и те же подзадачи много раз (пересекающиеся подзадачи).
- В рекурсивных решениях задач: Фибоначчи, рюкзак, расстояние редактирования и др.

---

## Преимущества

- **Рост производительности:** Снижает экспоненциальную сложность до полиномиальной.
- **Простота реализации:** Чаще всего добавляется всего несколько строк.

---

## Недостатки

- **Память:** Для больших входных данных может потребоваться много памяти для хранения результатов.

---

## Применение

- Задачи динамического программирования (Фибоначчи, рюкзак, монетная сдача и др.)
- Кэширование результатов в веб-серверах и API
- Оптимизация рекурсивных функций

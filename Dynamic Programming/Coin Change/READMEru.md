# Задача о размене монет: объяснение и пример

## Что такое задача о размене монет?

Задача о размене монет (Coin Change) — это классическая задача динамического программирования:  
Дано множество номиналов монет и сумма. Нужно найти минимальное количество монет, чтобы получить эту сумму.  
Если это невозможно — вернуть -1.

---

## Как работает DP-алгоритм для размена монет?

- **Состояние:** `dp[i]` — минимальное количество монет для суммы `i`.
- **Переход:** Для каждой монеты и для каждой суммы от `coin` до `amount`  
  обновляем: `dp[i] = min(dp[i], dp[i - coin] + 1)`.
- **Базовый случай:** `dp[0] = 0` (ноль монет для суммы 0).

Алгоритм поэтапно заполняет решение от 0 до нужной суммы.

---

## Пример на Python

```python
def coin_change(coins, amount):
    # Инициализация DP-массива
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0

    # Заполнение таблицы DP
    for coin in coins:
        for i in range(coin, amount + 1):
            if dp[i - coin] != float('inf'):
                dp[i] = min(dp[i], dp[i - coin] + 1)

    return dp[amount] if dp[amount] != float('inf') else -1

# Пример использования
coins = [1, 2, 5]
amount = 11
print(coin_change(coins, amount))  # Ответ: 3 (11 = 5 + 5 + 1)
```

---

## Сложность

- **Время:** O(n * amount), где n — количество номиналов монет
- **Память:** O(amount)

---

## Где используется?

- Системы размена в банкоматах и торговых автоматах
- Бюджетирование и распределение ресурсов
- Виртуальная валюта в видеоиграх

---

## Когда применять DP для размена монет?

- Когда нужен оптимальный (минимальный) размен
- Когда номиналы монет произвольные

---

## Реальный пример

Допустим, нужно собрать 11 из монет номиналом 1, 2 и 5. Алгоритм размена монет быстро подскажет минимальное число монет (3: 5 + 5 + 1).

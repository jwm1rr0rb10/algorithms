# Radix Sort (Поразрядная сортировка)

**Поразрядная сортировка (Radix Sort)** — это некомпаративный алгоритм сортировки, который сортирует числа, обрабатывая их цифры (или биты). Он работает путём многократного применения **стабильной сортировки** к каждой цифре числа — начиная с младшего разряда (LSD) и заканчивая старшим (MSD).

---

## Основная идея

- Поразрядная сортировка сортирует массив по каждой цифре, начиная с самой младшей (единицы) и двигаясь к самой старшей.
- Для каждой позиции используется **стабильный алгоритм сортировки** (например, сортировка подсчётом для цифр 0-9).
- Благодаря стабильности, элементы с одинаковой цифрой сохраняют порядок из предыдущих проходов, что даёт корректный общий порядок после всех проходов.

---

## Требования

- **Стабильный алгоритм сортировки** для сортировки по разряду (идеально — сортировка подсчётом для цифр 0-9).
- Элементы должны иметь "разряды" или "ключи", которые можно обрабатывать по отдельности (например, цифры числа, символы в строке фиксированной длины).

---

## Пример (по возрастанию, десятичные числа)

Массив: `[170, 45, 75, 90, 802, 24, 2, 66]`

**Шаг 1: Сортировка по единицам (младший разряд):**  
После стабильной сортировки по единицам: `[170, 90, 802, 2, 24, 45, 75, 66]`

**Шаг 2: Сортировка по десяткам:**  
После стабильной сортировки по десяткам: `[802, 2, 90, 24, 45, 66, 170, 75]`

**Шаг 3: Сортировка по сотням:**  
После стабильной сортировки по сотням: `[2, 24, 45, 66, 75, 90, 170, 802]`

Результат — отсортированный массив.

---

## ✅ Реализация на Go

```go
package main

import (
	"fmt"
)

// getMax находит максимальное значение в массиве
func getMax(arr []int) int {
	max := arr[0]
	for _, value := range arr {
		if value > max {
			max = value
		}
	}
	return max
}

// countSortForRadix выполняет сортировку подсчётом для заданного разряда (exp)
func countSortForRadix(arr []int, exp int) {
	n := len(arr)
	output := make([]int, n)
	count := make([]int, 10) // цифры 0-9

	for i := 0; i < n; i++ {
		digit := (arr[i] / exp) % 10
		count[digit]++
	}
	for i := 1; i < 10; i++ {
		count[i] += count[i-1]
	}
	for i := n - 1; i >= 0; i-- {
		digit := (arr[i] / exp) % 10
		pos := count[digit] - 1
		output[pos] = arr[i]
		count[digit]--
	}
	for i := 0; i < n; i++ {
		arr[i] = output[i]
	}
}

func radixSort(arr []int) {
	n := len(arr)
	if n <= 1 {
		return
	}
	m := getMax(arr)
	for exp := 1; m/exp > 0; exp *= 10 {
		countSortForRadix(arr, exp)
	}
}

func main() {
	data := []int{170, 45, 75, 90, 802, 24, 2, 66}
	fmt.Println("Исходный массив:", data)
	radixSort(data)
	fmt.Println("Отсортированный массив:", data)

	data2 := []int{4, 2, 7, 1, 9, 3, 6, 8, 5}
	fmt.Println("Исходный массив:", data2)
	radixSort(data2)
	fmt.Println("Отсортированный массив:", data2)

	data3 := []int{123, 45, 6, 7890, 1, 555}
	fmt.Println("Исходный массив:", data3)
	radixSort(data3)
	fmt.Println("Отсортированный массив:", data3)
}
```

---

## Разбор кода

- `getMax(arr []int) int`: Находит максимальное значение для определения числа разрядов.
- `countSortForRadix(arr []int, exp int)`: Стабильная сортировка подсчётом для цифры на позиции `exp`.
    - `(arr[i] / exp) % 10` извлекает цифру на текущем разряде.
    - Строит массив `count`, вычисляет накопительные суммы, заполняет выход в обратном порядке для стабильности.
    - Копирует результат обратно в исходный массив.
- `radixSort(arr []int)`: Организует сортировку, вызывая `countSortForRadix` для каждого разряда.

---

## Сложность поразрядной сортировки

- **Время:** O(nk), где n — количество элементов, k — количество разрядов в максимальном числе.
    - Для десятичной системы и разумных чисел k мало, сортировка близка к O(n).
- **Память:** O(n + b), где b — основание системы счисления (например, 10 для десятичных).

---

## Преимущества

- Может быть быстрее сортировок на сравнениях, если количество разрядов мало по сравнению с n.
- Стабильная сортировка (при стабильной внутренней сортировке).

## Недостатки

- Работает только для данных со структурой разрядов/ключей (числа, строки фиксированной длины).
- Требует стабильной внутренней сортировки.
- Не "на месте" (требуется вспомогательный массив).

## ✅ Реализация на Python

```python
# Вспомогательная: получить цифру на заданном разряде
def get_digit(number, exp):
    return (number // exp) % 10

def count_sort_for_radix(arr, exp):
    n = len(arr)
    output = [0] * n
    count = [0] * 10

    # Подсчёт цифр
    for i in range(n):
        digit = get_digit(arr[i], exp)
        count[digit] += 1
    # Накопление
    for i in range(1, 10):
        count[i] += count[i - 1]
    # Заполнение результата (с конца — для стабильности)
    for i in range(n - 1, -1, -1):
        digit = get_digit(arr[i], exp)
        pos = count[digit] - 1
        output[pos] = arr[i]
        count[digit] -= 1
    # Копирование обратно
    for i in range(n):
        arr[i] = output[i]

def radix_sort(arr):
    n = len(arr)
    if n <= 1:
        return
    max_val = max(arr)
    exp = 1
    while max_val // exp > 0:
        count_sort_for_radix(arr, exp)
        exp *= 10

# Примеры:
data_radix = [170, 45, 75, 90, 802, 24, 2, 66]
print("Исходный список (Radix Sort):", data_radix)
radix_sort(data_radix)
print("Отсортированный список (Radix Sort):", data_radix)

data_radix_2 = [4, 2, 7, 1, 9, 3, 6, 8, 5]
print("Исходный список (Radix Sort):", data_radix_2)
radix_sort(data_radix_2)
print("Отсортированный список (Radix Sort):", data_radix_2)

data_radix_3 = [123, 45, 6, 7890, 1, 555, 10000]
print("Исходный список (Radix Sort):", data_radix_3)
radix_sort(data_radix_3)
print("Отсортированный список (Radix Sort):", data_radix_3)
```

> _Поразрядная сортировка — мощный линейный алгоритм сортировки для целых чисел (и других типов с фиксированной "разрядной" структурой), если внутренняя сортировка стабильна. Она наглядно демонстрирует, как отказ от сравнений позволяет получить более быстрые алгоритмы для специализированных случаев._

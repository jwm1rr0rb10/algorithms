# Bucket Sort (Сортировка по корзинам)

**Сортировка по корзинам** — это алгоритм сортировки, не основанный на сравнениях, который работает путём распределения элементов исходного массива по «корзинам».

---

## Основная идея

1. Создать набор корзин (списков или массивов). Каждая корзина обычно соответствует определённому диапазону значений.
2. Пройтись по исходному массиву и поместить каждый элемент в соответствующую корзину на основе его значения.
3. Отсортировать элементы внутри каждой корзины отдельно (например, с помощью сортировки вставками, которая эффективна для небольших списков).
4. Объединить содержимое всех корзин по порядку, чтобы получить итоговый отсортированный массив.

---

## Требования

- Наилучшая производительность достигается, если входные данные равномерно распределены.
- Необходима функция отображения, которая определяет, в какую корзину поместить элемент.
- Для сортировки внутри корзин потребуется дополнительный алгоритм сортировки.

---

## Пример: Сортировка чисел с плавающей точкой в диапазоне от 0.0 до 1.0

**Входной массив:**  
`[0.78, 0.17, 0.39, 0.26, 0.72, 0.94, 0.21, 0.12, 0.23, 0.68]`

**Создадим 10 корзин для диапазонов:**  
`[0.0, 0.1), [0.1, 0.2), ..., [0.9, 1.0)`

**Шаг 1: Создать корзины**  
10 пустых списков: `buckets[0], buckets[1], ..., buckets[9]`

**Шаг 2: Распределить элементы по корзинам**  
Пример:  
```
0.78 -> buckets[7]
0.17 -> buckets[1]
0.39 -> buckets[3]
0.26 -> buckets[2]
0.72 -> buckets[7]
0.94 -> buckets[9]
0.21 -> buckets[2]
0.12 -> buckets[1]
0.23 -> buckets[2]
0.68 -> buckets[6]
```

**Шаг 3: Отсортировать элементы в каждой корзине (с помощью сортировки вставками или встроенной сортировки)**

- `buckets[1]: [0.12, 0.17]`
- `buckets[2]: [0.21, 0.23, 0.26]`
- `buckets[3]: [0.39]`
- `buckets[6]: [0.68]`
- `buckets[7]: [0.72, 0.78]`
- `buckets[9]: [0.94]`

**Шаг 4: Объединить корзины**  
Соединить все корзины по порядку.

**Результат:**  
`[0.12, 0.17, 0.21, 0.23, 0.26, 0.39, 0.68, 0.72, 0.78, 0.94]`

## ✅ Реализация на Go

```go
package main

import (
	"fmt"
	"sort"
)

// bucketSort сортирует список чисел с плавающей точкой в диапазоне от 0.0 до 1.0
func bucketSort(arr []float64, numBuckets int) {
	n := len(arr)
	if n == 0 || numBuckets <= 0 {
		return
	}

	// 1. Создать корзины
	buckets := make([][]float64, numBuckets)
	for i := range buckets {
		buckets[i] = []float64{}
	}

	// 2. Распределить элементы по корзинам
	for _, value := range arr {
		bucketIndex := int(value * float64(numBuckets))
		if bucketIndex == numBuckets {
			bucketIndex--
		}
		if bucketIndex < 0 || bucketIndex >= numBuckets {
			continue
		}
		buckets[bucketIndex] = append(buckets[bucketIndex], value)
	}

	// 3. Отсортировать корзины и 4. Объединить всё в исходный массив
	k := 0
	for i := 0; i < numBuckets; i++ {
		sort.Float64s(buckets[i])
		for _, value := range buckets[i] {
			arr[k] = value
			k++
		}
	}
}

func main() {
	data := []float64{0.78, 0.17, 0.39, 0.26, 0.72, 0.94, 0.21, 0.12, 0.23, 0.68}
	fmt.Println("Исходный массив (Bucket Sort):", data)
	bucketSort(data, 10)
	fmt.Println("Отсортированный массив (Bucket Sort):", data)

	data2 := []float64{0.5, 0.1, 0.9, 0.3, 0.7, 0.2, 0.8, 0.4, 0.6, 1.0}
	fmt.Println("Исходный массив (Bucket Sort):", data2)
	bucketSort(data2, 10)
	fmt.Println("Отсортированный массив (Bucket Sort):", data2)
}
```

## ✅ Реализация на Python

```python
def bucket_sort(arr, num_buckets):
    n = len(arr)
    if n == 0 or num_buckets <= 0:
        return

    # 1. Создать корзины
    buckets = [[] for _ in range(num_buckets)]

    # 2. Распределить элементы по корзинам
    for value in arr:
        bucket_index = int(value * num_buckets)
        if bucket_index == num_buckets:
            bucket_index -= 1
        if bucket_index < 0 or bucket_index >= num_buckets:
            continue
        buckets[bucket_index].append(value)

    # 3. Отсортировать каждую корзину и 4. Объединить их
    k = 0
    for i in range(num_buckets):
        buckets[i].sort()
        for value in buckets[i]:
            arr[k] = value
            k += 1

# Примеры использования Bucket Sort:
data_bucket = [0.78, 0.17, 0.39, 0.26, 0.72, 0.94, 0.21, 0.12, 0.23, 0.68]
print("Исходный список (Bucket Sort):", data_bucket)
bucket_sort(data_bucket, 10)
print("Отсортированный список (Bucket Sort):", data_bucket)

data_bucket_2 = [0.5, 0.1, 0.9, 0.3, 0.7, 0.2, 0.8, 0.4, 0.6, 1.0]
print("Исходный список (Bucket Sort):", data_bucket_2)
bucket_sort(data_bucket_2, 10)
print("Отсортированный список (Bucket Sort):", data_bucket_2)

data_bucket_3 = [0.05, 0.95, 0.001, 0.15, 0.85, 0.25, 0.75, 0.35, 0.65, 0.45]
print("Исходный список (Bucket Sort):", data_bucket_3)
bucket_sort(data_bucket_3, 5)
print("Отсортированный список (Bucket Sort):", data_bucket_3)
```

## Сложность

- **В среднем:** O(n + k), где n — количество элементов, k — количество корзин (предполагается равномерное распределение и маленькие корзины).
- **В худшем случае:** O(n²), если все элементы попадают в одну корзину (зависит от внутренней сортировки).
- **По памяти:** O(n + k)

---

## Преимущества

- Быстрый (O(n) в среднем случае), если входные данные равномерно распределены.
- Очень эффективен для чисел с плавающей точкой в известном диапазоне.

## Недостатки

- Производительность сильно зависит от распределения данных.
- Требуется дополнительная память под корзины.
- Необходим дополнительный алгоритм сортировки для каждой корзины.

---

> _Сортировка по корзинам — отличный пример того, как использование свойств ваших данных (таких как диапазон и распределение) позволяет добиться высокой эффективности сортировки в специальных случаях._

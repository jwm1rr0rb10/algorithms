# Insertion Sort (Сортировка вставками)

## 📋 Описание

**Сортировка вставками** — это простой алгоритм сортировки, который работает аналогично сортировке карт в руке. Из неотсортированной части массива по одному элементу берётся и вставляется в правильное место в уже отсортированной части, сдвигая большие элементы вправо.

### 🔑 Основная идея

- Массив делится на две части: отсортированную и неотсортированную.
- Первый элемент считается отсортированным.
- Каждый следующий элемент берётся из неотсортированной части и вставляется на своё место в отсортированной.

---

## 📈 Пример (по возрастанию)

Исходный массив:  
`[10, 7, 8, 9, 1, 5]`

Шаги сортировки:

1. `[10] | [7, 8, 9, 1, 5]` → `[7, 10] | [8, 9, 1, 5]`
2. `[7, 10] | [8, 9, 1, 5]` → `[7, 8, 10] | [9, 1, 5]`
3. `[7, 8, 10] | [9, 1, 5]` → `[7, 8, 9, 10] | [1, 5]`
4. `[7, 8, 9, 10] | [1, 5]` → `[1, 7, 8, 9, 10] | [5]`
5. `[1, 7, 8, 9, 10] | [5]` → `[1, 5, 7, 8, 9, 10]`

---

## 🧠 Сложность алгоритма

| Сценарий              | Временная сложность | Пространственная сложность |
|:----------------------|:-------------------|:--------------------------|
| Лучший (уже отсортирован) | O(n)            | O(1) (на месте)           |
| Средний               | O(n²)              | O(1)                      |
| Худший                | O(n²)              | O(1)                      |

## 🧪 Реализация

### ✅ Go

```go
package main

import "fmt"

// insertionSort сортирует массив с помощью сортировки вставками
func insertionSort(arr []int) {
    n := len(arr)
    for i := 1; i < n; i++ {
        key := arr[i]
        j := i - 1

        for j >= 0 && arr[j] > key {
            arr[j+1] = arr[j]
            j--
        }
        arr[j+1] = key
    }
}

func main() {
    data := []int{10, 7, 8, 9, 1, 5}
    fmt.Println("Исходный массив:", data)
    insertionSort(data)
    fmt.Println("Отсортированный массив:", data)

    data2 := []int{4, 2, 7, 1, 9, 3, 6, 8, 5}
    fmt.Println("Исходный массив:", data2)
    insertionSort(data2)
    fmt.Println("Отсортированный массив:", data2)
}
```

### 🐍 Python

```python
# insertion_sort сортирует список с помощью сортировки вставками
def insertion_sort(arr):
    n = len(arr)
    for i in range(1, n):
        key = arr[i]
        j = i - 1

        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

# Примеры использования сортировки вставками:
data_insertion = [10, 7, 8, 9, 1, 5]
print("\nИсходный список (Insertion Sort):", data_insertion)
insertion_sort(data_insertion)
print("Отсортированный список (Insertion Sort):", data_insertion)

data_insertion_2 = [4, 2, 7, 1, 9, 3, 6, 8, 5]
print("Исходный список (Insertion Sort):", data_insertion_2)
insertion_sort(data_insertion_2)
print("Отсортированный список (Insertion Sort):", data_insertion_2)
```

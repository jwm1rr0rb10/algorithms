# Heap Sort (Пирамидальная сортировка)

**Пирамидальная сортировка (Heap Sort)** — это алгоритм сортировки на сравнениях, который использует структуру данных под названием **двоичная куча**.

---

## Что такое двоичная куча?

**Двоичная куча** — это особый вид полного двоичного дерева с двумя основными свойствами:

### 1. Свойство полноты
- Все уровни дерева полностью заполнены, кроме, возможно, последнего.
- Последний уровень заполняется слева направо.
- Благодаря этому куча эффективно представляется в виде массива без пропусков.

Для элемента с индексом `i` в массиве:
- Левый потомок: `2*i + 1`
- Правый потомок: `2*i + 2`
- Родитель: `(i - 1) // 2` (целочисленное деление)

### 2. Свойство кучи
Существует два типа двоичных куч:
- **Макс-куча**: Значение любого узла больше или равно его потомкам. На корне — максимальный элемент.
- **Мин-куча**: Значение любого узла меньше или равно его потомкам. На корне — минимальный элемент.

Для сортировки по возрастанию обычно используют **макс-кучу**.

---

## Как работает Heap Sort

Heap Sort состоит из двух основных фаз:

### 1. Построение макс-кучи
Преобразовать исходный массив в макс-кучу. После этого максимальный элемент окажется на корне (индекс 0).

### 2. Сортировка кучи
- Обменять корень (максимум) с последним элементом.
- Уменьшить размер кучи на 1 (последний элемент теперь отсортирован).
- Восстановить свойство кучи операцией `heapify`, начиная с корня.
- Повторять до тех пор, пока размер кучи не станет равен 1.

В итоге массив отсортирован по возрастанию.

---

## Операция `heapify` (просеивание вниз)

Эта операция восстанавливает свойство кучи для поддерева с корнем в индексе `i`:

1. Считаем, что узел `i` — корень.
2. Находим наибольший элемент среди `i` и его потомков (в пределах размера кучи `n`).
3. Если один из потомков больше, чем `i`, меняем их местами.
4. Рекурсивно вызываем `heapify` для нового положения потомка.

---

## Пример

Дан массив: `[10, 7, 8, 9, 1, 5]` (n = 6)

### Фаза 1: Построение макс-кучи

Начинаем с последнего родителя: индекс `(n / 2) - 1 = 2` и движемся вверх.

- `heapify(arr, 6, 2)`
- `heapify(arr, 6, 1)`
- `heapify(arr, 6, 0)`

После этого массив становится (сохраняется структура кучи):  
`[10, 9, 8, 7, 1, 5]`

### Фаза 2: Сортировка

Обмениваем корень с последним элементом и делаем `heapify`:

```text
[10, 9, 8, 7, 1, 5] -> [5, 9, 8, 7, 1, 10] -> heapify -> [9, 7, 8, 5, 1, 10]
[9, 7, 8, 5, 1, 10] -> [1, 7, 8, 5, 9, 10] -> heapify -> [8, 7, 1, 5, 9, 10]
[8, 7, 1, 5, 9, 10] -> [5, 7, 1, 8, 9, 10] -> heapify -> [7, 5, 1, 8, 9, 10]
[7, 5, 1, 8, 9, 10] -> [1, 5, 7, 8, 9, 10] -> heapify -> [5, 1, 7, 8, 9, 10]
[5, 1, 7, 8, 9, 10] -> [1, 5, 7, 8, 9, 10] -> Готово.
```
Итоговый отсортированный массив:  
`[1, 5, 7, 8, 9, 10]`

## Реализация на Go

```go
package main

import "fmt"

// heapify восстанавливает свойство макс-кучи для поддерева с корнем в i.
func heapify(arr []int, n int, i int) {
	largest := i
	left := 2*i + 1
	right := 2*i + 2

	if left < n && arr[left] > arr[largest] {
		largest = left
	}
	if right < n && arr[right] > arr[largest] {
		largest = right
	}
	if largest != i {
		arr[i], arr[largest] = arr[largest], arr[i]
		heapify(arr, n, largest)
	}
}

// heapsort выполняет пирамидальную сортировку "на месте"
func heapsort(arr []int) {
	n := len(arr)
	// Построить макс-кучу
	for i := n/2 - 1; i >= 0; i-- {
		heapify(arr, n, i)
	}
	// Извлекать элементы по одному
	for i := n - 1; i > 0; i-- {
		arr[0], arr[i] = arr[i], arr[0]
		heapify(arr, i, 0)
	}
}

func main() {
	data := []int{10, 7, 8, 9, 1, 5}
	fmt.Println("Исходный массив:", data)
	heapsort(data)
	fmt.Println("Отсортированный массив:", data)

	data2 := []int{4, 2, 7, 1, 9, 3, 6, 8, 5}
	fmt.Println("Исходный массив:", data2)
	heapsort(data2)
	fmt.Println("Отсортированный массив:", data2)
}
```

---

## Разбор кода

### `heapify(arr []int, n int, i int)`
- Принимает массив, размер кучи `n`, и индекс `i` узла для восстановления свойства кучи.
- Вычисляет индексы левого и правого потомков.
- Сравнивает значение в `i` с потомками.
- При необходимости меняет местами и рекурсивно вызывает `heapify` для поддерева.

### `heapsort(arr []int)`
- Строит макс-кучу из массива.
- Проходит от последнего родителя до корня, вызывая `heapify`.
- Извлекает максимальный элемент (корень), помещает его в конец и восстанавливает кучу для оставшейся части.

---

## Сложность Heap Sort

### Временная сложность
- **Лучший / Средний / Худший случай:** O(n log n)
  - Построение кучи: O(n)
  - Каждый из n−1 проходов: O(log n)
  - **Всего:** O(n + n log n) = O(n log n)

### Пространственная сложность
- O(1) — сортировка на месте, требуется только константная дополнительная память.
- Глубина стека рекурсии в `heapify` — максимум O(log n).

---

## Преимущества пирамидальной сортировки

- Гарантированная сложность O(n log n) для любых входных данных.
- Алгоритм работает "на месте" (O(1) памяти).

## Недостатки

- Неустойчивая сортировка (равные элементы могут поменять порядок).
- Обычно медленнее быстрой сортировки из-за большего числа обменов и кэш-промахов.

---

## Когда использовать Heap Sort

- Когда важны гарантии по худшему времени.
- Когда требуется минимальное доп. потребление памяти.

## Реализация на Python

```python
# Функция для восстановления свойства макс-кучи для поддерева с корнем в i.
def heapify(arr, n, i):
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2

    if left < n and arr[left] > arr[largest]:
        largest = left

    if right < n and arr[right] > arr[largest]:
        largest = right

    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

# Основная функция сортировки кучей
def heapsort(arr):
    n = len(arr)
    # Построить макс-кучу
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    # Извлекать элементы по одному
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)

# Примеры использования:
data = [10, 7, 8, 9, 1, 5]
print("Исходный список:", data)
heapsort(data)
print("Отсортированный список:", data)

data2 = [4, 2, 7, 1, 9, 3, 6, 8, 5]
print("Исходный список:", data2)
heapsort(data2)
print("Отсортированный список:", data2)
```

# 🧼 Сортировка пузырьком (Bubble Sort)

**Сортировка пузырьком** — это простой алгоритм сортировки, который многократно проходит по списку, сравнивает соседние элементы и меняет их местами, если они идут в неправильном порядке. Этот процесс повторяется до тех пор, пока список не будет отсортирован. Более крупные элементы «всплывают» к концу списка с каждым проходом, как пузырьки в воде.

---

## 🔍 Идея

- Многократно проходить по массиву.
- На каждом проходе сравнивать соседние элементы.
- Если они стоят в неправильном порядке (например, первый больше второго при сортировке по возрастанию), поменять их местами.
- Повторять, пока не останется проходов без обменов — это означает, что массив отсортирован.

---

## 🧪 Пример (Сортировка по возрастанию)

Исходный массив:  
`[10, 7, 8, 9, 1, 5]`

### Проход 1:
- 10 > 7 → `[7, 10, 8, 9, 1, 5]`
- 10 > 8 → `[7, 8, 10, 9, 1, 5]`
- 10 > 9 → `[7, 8, 9, 10, 1, 5]`
- 10 > 1 → `[7, 8, 9, 1, 10, 5]`
- 10 > 5 → `[7, 8, 9, 1, 5, 10]`

### Проход 2:
- 7 < 8 → без обмена
- 8 < 9 → без обмена
- 9 > 1 → `[7, 8, 1, 9, 5, 10]`
- 9 > 5 → `[7, 8, 1, 5, 9, 10]`

### Проход 3:
- 7 < 8 → без обмена
- 8 > 1 → `[7, 1, 8, 5, 9, 10]`
- 8 > 5 → `[7, 1, 5, 8, 9, 10]`

### Проход 4:
- 7 > 1 → `[1, 7, 5, 8, 9, 10]`
- 7 > 5 → `[1, 5, 7, 8, 9, 10]`

### Проход 5:
- 1 < 5 → без обмена

Массив теперь отсортирован: `[1, 5, 7, 8, 9, 10]`

---

## ⚡ Оптимизация

Если полный проход завершился без единого обмена, алгоритм может завершиться досрочно — массив уже отсортирован.

---

## 📦 Временная и пространственная сложность

| Сложность             | Значение         |
|:--------------------- |:----------------|
| Время (лучший случай) | O(n)            |
| Время (ср./худший)    | O(n²)           |
| Память                | O(1) (на месте) |

## ✅ Реализация на Go

```go
package main

import "fmt"

// bubbleSort сортирует массив с помощью алгоритма пузырьковой сортировки
func bubbleSort(arr []int) {
	n := len(arr)
	// Внешний цикл проходит по всем элементам
	for i := 0; i < n-1; i++ {
		swapped := false // Флаг оптимизации: если обменов не было, массив уже отсортирован

		// Внутренний цикл сравнивает соседние элементы
		// (n-i-1) потому что последние i элементов уже на месте
		for j := 0; j < n-i-1; j++ {
			// Если текущий элемент больше следующего — меняем их местами
			if arr[j] > arr[j+1] {
				arr[j], arr[j+1] = arr[j+1], arr[j]
				swapped = true // Был совершен обмен
			}
		}

		// Если обменов не было — массив отсортирован
		if !swapped {
			break
		}
	}
}

func main() {
	data := []int{10, 7, 8, 9, 1, 5}
	fmt.Println("Исходный массив:", data)
	bubbleSort(data)
	fmt.Println("Отсортированный массив:", data)

	data2 := []int{1, 2, 3, 4, 5} // Уже отсортирован
	fmt.Println("Исходный массив:", data2)
	bubbleSort(data2)
	fmt.Println("Отсортированный массив:", data2)
}
```

## 🐍 Реализация на Python

```python
# bubble_sort сортирует список с помощью пузырьковой сортировки
def bubble_sort(arr):
    n = len(arr)
    # Внешний цикл проходит по всем элементам
    for i in range(n - 1):
        swapped = False  # Флаг оптимизации

        # Внутренний цикл сравнивает соседние элементы
        # (n - i - 1) потому что последние i элементов уже на месте
        for j in range(n - i - 1):
            # Если текущий элемент больше следующего — меняем их местами
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True  # Был совершен обмен

        # Если обменов не было — список отсортирован
        if not swapped:
            break

# Пример использования пузырьковой сортировки:
data_bubble = [10, 7, 8, 9, 1, 5]
print("Исходный список (Bubble Sort):", data_bubble)
bubble_sort(data_bubble)
print("Отсортированный список (Bubble Sort):", data_bubble)

data_bubble_2 = [1, 2, 3, 4, 5]  # Уже отсортирован
print("Исходный список (Bubble Sort):", data_bubble_2)
bubble_sort(data_bubble_2)
print("Отсортированный список (Bubble Sort):", data_bubble_2)
```

# Алгоритмы сортировки

В этом разделе представлен обзор и примеры реализации ключевых алгоритмов сортировки. Начнём с **быстрой сортировки (Quick Sort)** — классического и эффективного метода сортировки "разделяй и властвуй".

---

## 1. Быстрая сортировка (Quick Sort)

**Быстрая сортировка** — это эффективный алгоритм сортировки "на месте", использующий подход **разделяй и властвуй**. Разработан Тони Хоаром и широко используется благодаря своей производительности на практике.

### Основная идея

1. **Выбор опорного элемента:** Выбирается один элемент массива (первый, последний, средний или случайный). Выбор опорного элемента сильно влияет на эффективность.
2. **Разбиение (partition):** Массив перестраивается так, чтобы все элементы меньше опорного располагались слева от него, а большие — справа. Элементы, равные опорному, могут оказаться по любую сторону.
3. **Рекурсивное применение:** Быстрая сортировка рекурсивно применяется к подмассивам слева и справа от опорного.

Процесс останавливается, когда подмассив содержит 0 или 1 элемент (уже отсортирован).

---

### Шаг "Partition" (разбиение)

Ключевая часть быстрой сортировки. Существует несколько стратегий разбиения, здесь используется схема Ломуто — простая и популярная:

1. Опорный элемент — последний в массиве.
2. Указатель `i` отслеживает границу "меньших" элементов. Изначально `i = low - 1`.
3. Вторым указателем `j` проходим от `low` до `high - 1`.
4. Если `arr[j] <= pivot`:
    - Увеличиваем `i`.
    - Меняем `arr[i]` и `arr[j]` местами.
5. После цикла меняем местами `arr[i+1]` и опорный (`arr[high]`). Элемент на позиции `arr[i+1]` — на своём месте.

#### Пример (Ломуто, опорный — последний элемент):

Массив: `[10, 7, 8, 9, 1, 5]`, опорный элемент — `5`.

- `low = 0`, `high = 5`, `pivot = arr[5] = 5`
- `i = -1`

Проходим `j = 0..4`:

- `j=0`, `arr[0]=10` → 10 > 5, ничего не делаем, `i=-1`
- `j=1`, `arr[1]=7` → 7 > 5, ничего не делаем, `i=-1`
- `j=2`, `arr[2]=8` → 8 > 5, ничего не делаем, `i=-1`
- `j=3`, `arr[3]=9` → 9 > 5, ничего не делаем, `i=-1`
- `j=4`, `arr[4]=1` → 1 <= 5:
    - `i=0`
    - swap `arr[0]` и `arr[4]`: `[1, 7, 8, 9, 10, 5]`

После цикла меняем `arr[1]` (7) и `arr[5]` (5):
Результат: `[1, 5, 8, 9, 10, 7]`

Теперь опорный `5` на позиции 1. Всё слева (`[1]`) ≤ 5, всё справа (`[8, 9, 10, 7]`) > 5.

Далее рекурсивно сортируем `[1]` и `[8, 9, 10, 7]`.

### Реализация на Go

```go
package main

import "fmt"

func quicksort(arr []int) {
	quicksortRecursive(arr, 0, len(arr)-1)
}

func quicksortRecursive(arr []int, low, high int) {
	if low < high {
		pi := partition(arr, low, high)
		quicksortRecursive(arr, low, pi-1)
		quicksortRecursive(arr, pi+1, high)
	}
}

func partition(arr []int, low, high int) int {
	pivot := arr[high]
	i := low - 1
	for j := low; j < high; j++ {
		if arr[j] <= pivot {
			i++
			arr[i], arr[j] = arr[j], arr[i]
		}
	}
	arr[i+1], arr[high] = arr[high], arr[i+1]
	return i + 1
}

func main() {
	data := []int{10, 7, 8, 9, 1, 5}
	fmt.Println("Исходный массив:", data)
	quicksort(data)
	fmt.Println("Отсортированный массив:", data)

	data2 := []int{4, 2, 7, 1, 9, 3, 6, 8, 5}
	fmt.Println("Исходный массив:", data2)
	quicksort(data2)
	fmt.Println("Отсортированный массив:", data2)
}
```

---

### Разбор кода

- `quicksort(arr []int)`: Вспомогательная функция, вызывает `quicksortRecursive` для всего массива.
- `quicksortRecursive(arr []int, low, high int)`: Основная рекурсивная функция. Если `low < high`, делит массив и рекурсивно сортирует части.
- `partition(arr []int, low, high int) int`: Реализует разбиение по Ломуто. Возвращает индекс опорного элемента.

### Реализация на Python

```python
# Функция разбиения массива
def partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

# Основная рекурсивная функция быстрой сортировки
def quicksort(arr, low, high):
    if low < high:
        pi = partition(arr, low, high)
        quicksort(arr, low, pi - 1)
        quicksort(arr, pi + 1, high)

# Вспомогательная функция для сортировки всего массива
def sort_quick(arr):
    quicksort(arr, 0, len(arr) - 1)

# Примеры использования:
data = [10, 7, 8, 9, 1, 5]
print("Исходный массив:", data)
sort_quick(data)
print("Отсортированный массив:", data)

data2 = [4, 2, 7, 1, 9, 3, 6, 8, 5]
print("Исходный массив:", data2)
sort_quick(data2)
print("Отсортированный массив:", data2)
```

### Сложность быстрой сортировки

**Время:**
- **Средний случай:** O(n log n) — если опорный делит массив примерно пополам.
- **Худший случай:** O(n²) — если опорный всегда минимальный или максимальный элемент (например, отсортированный массив и неудачный выбор опорного).

**Память:**
- **В среднем:** O(log n) (рекурсивный стек).
- **В худшем случае:** O(n) (глубокая рекурсия при неудачном выборе опорного).

**Преимущества:**
- Быстрая на практике.
- "На месте" (in-place), требует минимум доп. памяти (кроме стека вызовов).

**Недостатки:**
- Худший случай — O(n²).
- Неустойчива (относительный порядок равных элементов может нарушаться).

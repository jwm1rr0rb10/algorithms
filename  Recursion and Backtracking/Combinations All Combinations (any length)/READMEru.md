# Все сочетания любой длины (All Combinations / Subsets) — Рекурсия и бэктрекинг

## Описание задачи

Дан массив чисел (или диапазон от 1 до n).  
Требуется вернуть **все возможные комбинации (подмножества) любого размера** — от пустого до полного массива.

**Пример:**  
Ввод: [1, 2, 3]  
Вывод:  
```
[
  [],
  [1],
  [2],
  [3],
  [1,2],
  [1,3],
  [2,3],
  [1,2,3]
]
```

---

## Идея и подход

- Это классическая задача на **генерацию всех подмножеств** (power set).
- Для каждого элемента есть два варианта: включить его в текущее подмножество или нет.
- Используем рекурсию и бэктрекинг: на каждом шаге добавляем текущий набор (path) в ответ, затем пробуем добавить следующий элемент.

---

## Пример на Python

```python
def subsets(nums):
    result = []

    def backtrack(start, path):
        result.append(list(path))
        for i in range(start, len(nums)):
            path.append(nums[i])
            backtrack(i + 1, path)
            path.pop()  # бэктрекинг

    backtrack(0, [])
    return result

# Пример использования:
print(subsets([1, 2, 3]))
# [[], [1], [2], [3], [1, 2], [1, 3], [2, 3], [1, 2, 3]]
```

---

## Как это работает

1. Начинаем с пустого набора (path) и позиции start = 0.
2. Добавляем текущий path в результат (даже если он пустой).
3. Для каждого i от start до конца:
    - Добавляем nums[i] к path.
    - Рекурсивно ищем подмножества для следующего индекса.
    - После возврата удаляем последний элемент (бэктрекинг).

---

## Сложность

- **Время:** O(2^n * n), где n — длина массива (всего 2^n подмножеств, каждое может иметь до n элементов).
- **Память:** O(2^n * n) — на хранение всех подмножеств.

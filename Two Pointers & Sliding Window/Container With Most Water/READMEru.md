# Контейнер с максимальным количеством воды — подход двух указателей

## Описание задачи

Дан массив целых чисел `height`, где каждый элемент — это высота вертикальной линии на оси x. Найдите такие две линии, которые вместе с осью x образуют контейнер, способный вместить максимальное количество воды. Верните максимальную площадь, которую может вместить такой контейнер.

**Пример:**  
Ввод: height = [1,8,6,2,5,4,8,3,7]  
Вывод: 49  
Пояснение: Линии с индексами 1 и 8 (высоты 8 и 7) образуют контейнер с площадью 7 * 7 = 49 (ширина 7, высота 7).

---

## Идея и подход алгоритма

- Используйте два указателя: один в начале (`left`), другой в конце (`right`) массива.
- На каждом шаге вычисляйте площадь между линиями:  
  Площадь = min(height[left], height[right]) * (right - left)
- Обновляйте максимальную найденную площадь.
- Передвигайте тот указатель, который указывает на меньшую высоту (есть шанс найти больший контейнер).
- Продолжайте, пока указатели не встретятся.

---

## Пример на Python

```python
def maxArea(height):
    left, right = 0, len(height) - 1
    max_area = 0

    while left < right:
        h = min(height[left], height[right])
        w = right - left
        max_area = max(max_area, h * w)
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    return max_area

# Пример использования:
print(maxArea([1,8,6,2,5,4,8,3,7]))  # Выведет: 49
```

---

## Анализ сложности

- **Время:** O(n), где n — длина массива (каждый указатель двигается максимум n раз).
- **Память:** O(1), используются только несколько переменных.

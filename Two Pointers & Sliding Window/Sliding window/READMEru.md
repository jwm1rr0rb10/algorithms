# Скользящее окно (Sliding Window)

## Описание

**Скользящее окно** — это алгоритмический приём, который используется для эффективной обработки последовательностей (массивов, строк) с помощью динамического подмассива (или "окна") фиксированного или переменного размера.

Этот подход позволяет находить решения для задач на подмассивы/подстроки без необходимости многократно пересчитывать значения для каждого возможного интервала.

---

## Как это работает

1. **Начальное окно:**  
   Определяется начальный диапазон элементов (например, первые `k` элементов массива).
2. **Сдвиг окна:**  
   Окно "скользит" по последовательности: крайний левый элемент удаляется, а новый элемент добавляется справа.
3. **Обновление результата:**  
   После каждого сдвига результат пересчитывается только для изменённых элементов, что значительно ускоряет работу алгоритма.

---

## Пример задачи

> Найти максимальную сумму подмассива длины `k` в массиве.

### Пример кода на Python

```python
def max_sum_subarray(arr, k):
    window_sum = sum(arr[:k])
    max_sum = window_sum

    for i in range(k, len(arr)):
        window_sum += arr[i] - arr[i - k]
        max_sum = max(max_sum, window_sum)

    return max_sum
```

---

## Когда использовать скользящее окно

- Поиск максимума/минимума/суммы в подмассиве фиксированной длины.
- Поиск подстроки с определёнными свойствами в строке.
- Оптимизация задач, где требуется обработка всех подмассивов/подстрок размера `k`.

---

## Преимущества

- **Скорость:** Избегает повторного пересчёта для каждого окна.
- **Простота:** Легко реализуется.
- **Универсальность:** Применим к множеству задач на последовательности.

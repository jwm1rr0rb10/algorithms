# Переместить нули (Move Zeroes)

## Описание задачи

Дан массив целых чисел `nums`. Необходимо переместить все нули в конец массива, сохранив относительный порядок остальных (ненулевых) элементов. Сделать это нужно на месте (in-place) с минимальным количеством операций.

**Пример:**  
Вход: nums = [0,1,0,3,12]  
Выход: [1,3,12,0,0]  
Пояснение: Все ненулевые элементы перемещены в начало, а нули — в конец.

---

## Идея алгоритма и подход

- Используется техника двух указателей: один указывает, куда ставить следующий ненулевой элемент (`last_non_zero`), второй проходит по массиву.
- В первом проходе все ненулевые элементы сдвигаются в начало.
- Во втором проходе оставшиеся позиции заполняются нулями.

---

## Пример на Python

```python
def moveZeroes(nums):
    last_non_zero = 0
    # Сдвигаем ненулевые элементы вперёд
    for i in range(len(nums)):
        if nums[i] != 0:
            nums[last_non_zero] = nums[i]
            last_non_zero += 1
    # Заполняем оставшиеся позиции нулями
    for i in range(last_non_zero, len(nums)):
        nums[i] = 0

# Пример использования:
arr = [0,1,0,3,12]
moveZeroes(arr)
print(arr)  # Вывод: [1, 3, 12, 0, 0]
```

---

## Анализ сложности

- **Время:** O(n), где n — длина массива (два прохода по массиву).
- **Память:** O(1), всё делается на месте.

## Применение в жизни

- Очистка данных: перенос пустых или отсутствующих значений (нулей) в конец для удобства обработки.
- Перераспределение списков для эффективных вычислений.
- Оптимизация доступа к памяти: группировка активных элементов в начале массива.

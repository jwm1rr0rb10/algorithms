# Удаление дубликатов из отсортированного массива

## Описание задачи

Дан отсортированный массив `nums`. Необходимо удалить дубликаты так, чтобы каждый уникальный элемент встречался только один раз. Итоговый порядок элементов должен сохраняться.  
После удаления дубликатов вернуть количество уникальных элементов. Необходимо выполнить это на месте (in-place) с памятью O(1).

**Пример:**  
Вход: nums = [1,1,2]  
Выход: 2, nums = [1,2,_]  
Пояснение: Функция должна вернуть length = 2, а первые два элемента массива должны быть 1 и 2.  
То, что находится дальше по массиву, неважно.

---

## Идея алгоритма и подход

- Используется техника двух указателей:
  - `i` указывает на позицию последнего уникального элемента.
  - `j` пробегает по массиву.
- Для каждого элемента, если `nums[j]` не равен `nums[i]`, увеличиваем `i` и записываем `nums[i] = nums[j]`.
- После окончания цикла первые `i + 1` элементов массива будут уникальными.

---

## Пример на Python

```python
def removeDuplicates(nums):
    if not nums:
        return 0
    i = 0
    for j in range(1, len(nums)):
        if nums[j] != nums[i]:
            i += 1
            nums[i] = nums[j]
    return i + 1

# Пример использования:
arr = [1,1,2]
length = removeDuplicates(arr)
print(length)        # Вывод: 2
print(arr[:length])  # Вывод: [1, 2]
```

---

## Анализ сложности

- **Время:** O(n), где n — длина массива (один проход).
- **Память:** O(1), всё делается на месте.

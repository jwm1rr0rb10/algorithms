# Two Sum (Два числа в сумме)

## Описание задачи

Дан массив целых чисел `nums` и целое число `target`. Необходимо вернуть индексы двух чисел, сумма которых равна `target`.

Можно предположить, что всегда существует ровно одно решение, и один элемент нельзя использовать дважды.

**Пример:**  
Вход: nums = [2,7,11,15], target = 9  
Выход: [0,1]  
Пояснение: nums[0] + nums[1] == 9, поэтому возвращаем [0, 1].

---

## Идея алгоритма и подход

- Используем хеш-таблицу (словарь) для хранения чисел и их индексов по мере прохода по массиву.
- Для каждого элемента `x` проверяем, существует ли в таблице число `target - x`.
  - Если да — нашли пару.
  - Если нет — добавляем `x` и его индекс в таблицу.

---

## Пример на Python

```python
def twoSum(nums, target):
    lookup = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in lookup:
            return [lookup[complement], i]
        lookup[num] = i

# Пример использования:
print(twoSum([2,7,11,15], 9))  # Вывод: [0, 1]
```

---

## Анализ сложности

- **Время:** O(n), где n — длина массива (один проход).
- **Память:** O(n), используется хеш-таблица.

## Применение в жизни

- Поиск пар транзакций или значений, сумма которых равна нужной (финансовые задачи).
- Парное сопоставление ресурсов или задач.
- Используется как подзадача в более сложных алгоритмах (например, 3Sum, 4Sum).

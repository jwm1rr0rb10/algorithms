# Сортировка цветов (Задача о голландском флаге)

## Описание задачи

Дан массив `nums`, состоящий из n элементов, окрашенных в красный, белый или синий (обозначено числами 0, 1 и 2). Нужно отсортировать их на месте так, чтобы одинаковые цвета стояли рядом, в порядке: красный (0), белый (1), синий (2).

Использовать встроенные функции сортировки нельзя.

**Пример:**  
Вход: nums = [2,0,2,1,1,0]  
Выход: [0,0,1,1,2,2]  
Пояснение: Массив отсортирован в порядке [0,0,1,1,2,2].

---

## Идея алгоритма и подход

- Используется алгоритм Дейкстры (голландский флаг) с тремя указателями:
  - `low` — позиция для следующего 0.
  - `mid` — текущий рассматриваемый элемент.
  - `high` — позиция для следующего 2.
- Проходим по массиву с помощью `mid`:
  - Если `nums[mid] == 0`: меняем местами с `nums[low]`, увеличиваем `low` и `mid`.
  - Если `nums[mid] == 1`: просто увеличиваем `mid`.
  - Если `nums[mid] == 2`: меняем с `nums[high]`, уменьшаем `high` (но `mid` не увеличиваем).

---

## Пример на Python

```python
def sortColors(nums):
    low, mid, high = 0, 0, len(nums) - 1
    while mid <= high:
        if nums[mid] == 0:
            nums[low], nums[mid] = nums[mid], nums[low]
            low += 1
            mid += 1
        elif nums[mid] == 1:
            mid += 1
        else:  # nums[mid] == 2
            nums[mid], nums[high] = nums[high], nums[mid]
            high -= 1

# Пример использования:
arr = [2,0,2,1,1,0]
sortColors(arr)
print(arr)  # Вывод: [0, 0, 1, 1, 2, 2]
```

---

## Анализ сложности

- **Время:** O(n), один проход по массиву.
- **Память:** O(1), сортировка на месте.

## Применение в жизни

- Быстрая сортировка объектов по категориям в один проход (например, по цвету или приоритету).
- Разделение данных для ускорения доступа и обработки.
- Используется в вариантах быстрой сортировки и других in-place алгоритмах.

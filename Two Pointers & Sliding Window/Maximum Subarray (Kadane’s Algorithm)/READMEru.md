# Максимальная сумма подмассива — алгоритм Кадане

## Описание задачи

Дан массив целых чисел `nums`. Найдите такой непрерывный подмассив (содержащий хотя бы одно число), сумма которого максимальна, и верните эту сумму.

**Пример:**  
Ввод: nums = [-2,1,-3,4,-1,2,1,-5,4]  
Вывод: 6  
Пояснение: Подмассив [4,-1,2,1] имеет максимальную сумму 6.

---

## Идея и подход алгоритма

- Использовать алгоритм Кадане для эффективного поиска максимальной суммы непрерывного подмассива.
- Завести две переменные:  
  - `current_sum` — максимальная сумма на текущей позиции (локальный максимум).
  - `max_sum` — максимальная сумма среди всех просмотренных (глобальный максимум).
- Для каждого элемента массива:
  - Обновить `current_sum` как максимум между текущим элементом и `current_sum + текущий элемент`.
  - Обновить `max_sum`, если `current_sum` больше текущего `max_sum`.

---

## Пример на Python

```python
def maxSubArray(nums):
    current_sum = max_sum = nums[0]
    for num in nums[1:]:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)
    return max_sum

# Пример использования:
print(maxSubArray([-2,1,-3,4,-1,2,1,-5,4]))  # Выведет: 6
```

---

## Анализ сложности

- **Время:** O(n), где n — количество элементов в массиве (каждый элемент рассматривается один раз).
- **Память:** O(1), используются только две переменные.

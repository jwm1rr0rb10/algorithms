# 🔍 Хеширование (использование HashMap / HashSet)

## 📌 Описание задачи

Хеширование — фундаментальное понятие в информатике, заключающееся в отображении данных произвольного размера в значение фиксированного размера, как правило, целое число, называемое хеш-кодом или хеш-значением. Этот код затем используется для быстрой локализации или хранения данных.

HashMap (также известный как хеш-таблица или словарь) и HashSet — это структуры данных, реализующие концепцию хеширования, обеспечивая высокоэффективную среднюю производительность для таких операций, как вставка, удаление и поиск.

* **HashMap**: Хранит пары ключ-значение. По ключу можно очень быстро получить соответствующее значение. Ключи должны быть уникальными, значения могут повторяться.
* **HashSet**: Хранит только уникальные элементы. Фактически, это HashMap, где хранятся только ключи, а значения — фиктивные. Используется для быстрого определения наличия элемента.

## 💡 Идея и подход

Основная идея HashMap и HashSet — использование хеш-функции для преобразования ключа (или элемента) в индекс в базовом массиве (часто называемом «массив корзин» или «таблицей»).

### Как это работает

1. **Хеш-функция**: Вычисляет целочисленный хеш-код из ключа. Хорошая хеш-функция равномерно распределяет ключи, минимизируя коллизии.
2. **Расчёт индекса**: Хеш-код обычно делится по модулю размера массива для определения нужной корзины.
3. **Разрешение коллизий**:

   * **Разделённые цепочки** (Separate Chaining): Каждая корзина хранит список или дерево элементов.
   * **Открытая адресация** (Open Addressing): Использует пробинг (например, линейный, квадратичный) для поиска пустой ячейки.

При получении значения (или проверке наличия) применяется та же хеш-функция и алгоритм расчёта индекса.

## 🧪 Пример на Python

```python
# --- HashMap (словарь Python) ---
print("--- Пример HashMap ---")
my_map = {}

my_map["apple"] = 10
my_map["banana"] = 20
my_map["cherry"] = 30
my_map["date"] = 10  # Повторяющееся значение

print(f"Карта после вставок: {my_map}")

print(f"Значение для 'banana': {my_map['banana']}")
print(f"Значение для 'apple': {my_map.get('apple', 'Не найдено')}")
print(f"Значение для 'grape': {my_map.get('grape', 'Не найдено')}")

print(f"'cherry' в карте: {'cherry' in my_map}")
print(f"'fig' в карте: {'fig' in my_map}")

del my_map["banana"]
print(f"Карта после удаления 'banana': {my_map}")

for key in my_map:
    print(f"Ключ: {key}, Значение: {my_map[key]}")

for key, value in my_map.items():
    print(f"Ключ: {key}, Значение: {value}")

# --- HashSet (множество Python) ---
print("\n--- Пример HashSet ---")
my_set = set()

my_set.add("red")
my_set.add("green")
my_set.add("blue")
my_set.add("red")  # Повтор игнорируется

print(f"Множество после добавлений: {my_set}")

print(f"'green' в множестве: {'green' in my_set}")
print(f"'yellow' в множестве: {'yellow' in my_set}")

my_set.remove("blue")
print(f"Множество после удаления 'blue': {my_set}")

my_set.discard("yellow")
print(f"Множество после удаления 'yellow': {my_set}")

set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

print(f"Объединение: {set1.union(set2)}")
print(f"Пересечение: {set1.intersection(set2)}")
print(f"Разность: {set1.difference(set2)}")
print(f"Симметричная разность: {set1.symmetric_difference(set2)}")

print("Итерирование по элементам множества:")
for item in my_set:
    print(item)
```

## ⏱️ Сложность

### Средняя временная сложность:

* Вставка: `O(1)`
* Удаление: `O(1)`
* Поиск: `O(1)`

### Худший случай:

* Все операции: `O(n)`

### Пространственная сложность:

* `O(n)` для хранения `n` элементов.

## ⚠️ Особенности

* **Изменяемые ключи**: В Python ключами могут быть только неизменяемые типы (int, str, tuple).
* **Хорошая хеш-функция**: Должна быть быстрой и обеспечивать равномерное распределение.
* **Коэффициент загрузки**: При высоком значении производительность может ухудшаться, вызывая перераспределение (rehashing).
* **Худшие случаи**: Важно учитывать `O(n)` в случае множества коллизий.

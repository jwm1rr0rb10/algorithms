# 📏 НОД — Алгоритм Евклида

## 📌 Описание задачи

**Наибольший общий делитель (НОД)** двух чисел `a` и `b` — это наибольшее число, которое делит оба без остатка.  
**Алгоритм Евклида** эффективно вычисляет `gcd(a, b)` по формуле:

- `gcd(a, b) = gcd(b, a % b)`


Рекурсия продолжается, пока `b = 0`, тогда `gcd(a, 0) = a`.

---

## 💡 Идея и подход

Алгоритм основан на том, что:

- Если `a = bq + r`, то `gcd(a, b) = gcd(b, r)`
- Это уменьшает размер задачи на каждом шаге

---

## 🧪 Пример на Python

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

# Пример
print(gcd(30, 18))  # Вывод: 6
```

---

## ⏱️ Сложность
- Время: O(log min(a, b))
- Память: O(1) (в итеративной версии)

---

## 🧭 Применение
- Сокращение дробей: a / b → делим числитель и знаменатель на gcd(a, b)
- Обратный по модулю (через расширенный алгоритм Евклида)
- НОК: lcm(a, b) = a * b / gcd(a, b)
- Теория чисел, криптография, диофантовы уравнения

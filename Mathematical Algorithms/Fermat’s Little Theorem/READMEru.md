# 🔁 Малая теорема Ферма

## 📌 Описание задачи

**Малая теорема Ферма** утверждает:

> Если `p` — простое число, и `a` — целое число, не делящееся на `p`, то:
>
> ```
> a^(p - 1) ≡ 1 mod p
> ```

Из этого следует формула для нахождения обратного по модулю:

- `a^(-1) ≡ a^(p - 2) mod p`


Это один из самых эффективных способов найти обратный элемент по модулю, если модуль — простое число.

---

## 💡 Идея и подход

Чтобы найти `a^(-1) mod p`:

1. Убедиться, что `p` — простое, и `gcd(a, p) = 1`
2. Вычислить `a^(p - 2) % p` с помощью быстрого возведения в степень

---

## 🧪 Пример на Python

```python
def mod_inverse(a, p):
    return pow(a, p - 2, p)  # Малая теорема Ферма

# Пример
print(mod_inverse(3, 7))  # Вывод: 5, так как 3 * 5 ≡ 1 mod 7
```

---

## ⏱️ Сложность
- Время: O(log p) (через fast power)
- Память: O(1)

---

## 🧭 Применение
- Обратные элементы в комбинаторике (nCr % p)
- Криптография (RSA, Diffie-Hellman)
- Решение уравнений по модулю
- Сокращение дробей в модульной арифметике

---

## ⚠️ Ограничения
- Работает только если mod — простое число
- Если модуль не простой — использовать расширенный алгоритм Евклида

# 🧮 Китайская теорема об остатках (Chinese Remainder Theorem)

## 📌 Описание задачи

**Китайская теорема об остатках (КТО)** утверждает, что для системы линейных сравнений с попарно взаимно простыми модулями существует единственное решение по модулю произведения этих модулей. Если дана система:

- x ≡ a₁ mod m₁ 
- x ≡ a₂ mod m₂ 
- ... 
- x ≡ aₖ mod mₖ

- где `m₁, m₂, ..., mₖ` — попарно взаимно простые положительные числа,  
то существует единственное решение `x` по модулю `M`, где:

- M = m₁ × m₂ × ... × mₖ


---

## 💡 Идея и подход

Чтобы найти решение, выполняем следующие шаги:

1. **Вычислить M**:  
   `M = m₁ × m₂ × ... × mₖ`

2. **Вычислить Mᵢ**:  
   Для каждого `i`: `Mᵢ = M / mᵢ`

3. **Найти обратный элемент yᵢ**:  
   Для каждого `i` найти `yᵢ`, такое что:  
   `Mᵢ × yᵢ ≡ 1 mod mᵢ`  
   Это делается с помощью расширенного алгоритма Евклида.

4. **Вычислить результат**:  
   Итоговое решение:

- x ≡ (a₁ × M₁ × y₁ + a₂ × M₂ × y₂ + ... + aₖ × Mₖ × yₖ) mod M


---

## 🧪 Пример на Python

```python
def extended_gcd(a, b):
 if a == 0:
     return b, 0, 1
 gcd, x1, y1 = extended_gcd(b % a, a)
 x = y1 - (b // a) * x1
 y = x1
 return gcd, x, y

def mod_inverse(a, m):
 gcd, x, _ = extended_gcd(a, m)
 if gcd != 1:
     raise Exception("Обратного элемента не существует")
 return (x % m + m) % m

def chinese_remainder_theorem(congruences):
 # congruences: список пар (aᵢ, mᵢ)
 M = 1
 for _, m in congruences:
     M *= m

 result = 0
 for a_i, m_i in congruences:
     M_i = M // m_i
     y_i = mod_inverse(M_i, m_i)
     result += a_i * M_i * y_i

 return result % M

# Пример
congruences = [(2, 3), (3, 5), (2, 7)]
print(chinese_remainder_theorem(congruences))  # Вывод: 23
```
--- 

## ✅ Проверка:
- 23 % 3 = 2
- 23 % 5 = 3
- 23 % 7 = 2

## ⏱️ Сложность
- Время:
    - Вычисление M: O(k)
    - Для каждой пары: нахождение обратного элемента через расширенный алгоритм Евклида → O(log mᵢ)
    - Итого: O(k log M)

- Память:
    - O(k) — для хранения промежуточных значений

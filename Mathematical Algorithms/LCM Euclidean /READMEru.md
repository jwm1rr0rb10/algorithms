# 🔁 НОК — Наименьшее общее кратное (через алгоритм Евклида)

## 📌 Описание задачи

**Наименьшее общее кратное (НОК)** двух чисел `a` и `b` — это наименьшее положительное число, которое делится и на `a`, и на `b`.

НОК связан с НОД по формуле:

- `lcm(a, b) = abs(a * b) / gcd(a, b)`


Это позволяет эффективно вычислять НОК через алгоритм Евклида.

---

## 💡 Идея и подход

1. Вычислить `gcd(a, b)` через алгоритм Евклида
2. Использовать формулу:  
   `lcm(a, b) = abs(a * b) // gcd(a, b)`

---

## 🧪 Пример на Python

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

def lcm(a, b):
    return abs(a * b) // gcd(a, b)

# Пример
print(lcm(12, 18))  # Вывод: 36
```

---

## ⏱️ Сложность
- Время: O(log min(a, b))
- Память: O(1)

--- 

## 🧭 Применение
- Синхронизация циклов (например, светофоры, шестерёнки)
- Задачи на расписания
- Теория чисел и модульная арифметика
- Криптография (например, RSA использует lcm(φ(p), φ(q)))

---

## ⚠️ Осторожно
- Используйте abs(a * b), чтобы избежать отрицательных значений
- При больших a и b используйте a // gcd(a, b) * b, чтобы избежать переполнения

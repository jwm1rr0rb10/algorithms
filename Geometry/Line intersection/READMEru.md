# 📐 Пересечение прямых: Объяснение и пример

## Что такое задача пересечения прямых?

Задача **пересечения прямых** формулируется так:  
Даны две бесконечные прямые на плоскости. Определить, пересекаются ли они.  
Если да — найти точку пересечения.

В отличие от отрезков, прямые считаются бесконечными в обе стороны, поэтому пересечение всегда определено, если они не параллельны или совпадающие.

---

## 📌 Зачем это нужно?

- Используется в рендеринге, трассировке лучей и моделировании  
- Является основой аналитической геометрии и линейной алгебры  
- Применяется в робототехнике, физических движках и CAD-системах

---

## ⚙️ Как работает алгоритм?

### Шаг 1: Представление прямых в общем виде

Каждая прямая задаётся двумя точками:  
- Прямая 1: (x₁, y₁) → (x₂, y₂)  
- Прямая 2: (x₃, y₃) → (x₄, y₄)

Преобразуем каждую в форму:  
**Ax + By = C**

### Шаг 2: Решение системы уравнений

Используем определитель (правило Крамера):

`A1 * x + B1 * y = C1 A2 * x + B2 * y = C2`


- Если определитель равен нулю → прямые параллельны или совпадают  
- Иначе → вычисляем точку пересечения (x, y)

---

## 🧪 Пример на Python

```python
def line_intersection(p1, p2, p3, p4):
    # Прямая 1: p1 → p2
    A1 = p2[1] - p1[1]
    B1 = p1[0] - p2[0]
    C1 = A1 * p1[0] + B1 * p1[1]

    # Прямая 2: p3 → p4
    A2 = p4[1] - p3[1]
    B2 = p3[0] - p4[0]
    C2 = A2 * p3[0] + B2 * p3[1]

    # Определитель
    det = A1 * B2 - A2 * B1

    if det == 0:
        return None  # Прямые параллельны или совпадают

    # Правило Крамера
    x = (C1 * B2 - C2 * B1) / det
    y = (A1 * C2 - A2 * C1) / det
    return (x, y)

# Пример использования
A = (1, 1)
B = (4, 4)
C = (1, 4)
D = (4, 1)
print(line_intersection(A, B, C, D))  # Вывод: (2.5, 2.5)
```

---

## ⏱️ Сложность
- Время: O(1)
- Память: O(1)

## 🧭 Применение
- Трассировка лучей и рендеринг
- Аналитическая геометрия
- Пересечение дорог, траекторий, маршрутов
- Решение систем линейных уравнений

---

## ✅ Итог
- Пересечение прямых вычисляет точку, в которой пересекаются две бесконечные прямые.
- Используется общее уравнение прямой и правило Крамера.
- Не проверяет, лежит ли точка на отрезке — предполагается, что прямые бесконечны.

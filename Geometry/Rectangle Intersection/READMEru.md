# Пересечение прямоугольников: Объяснение и пример

## Что такое задача пересечения прямоугольников?

Задача **пересечения прямоугольников** формулируется так:  
Даны два прямоугольника на плоскости (выравненные по осям). Нужно определить, пересекаются ли они.  
Дополнительно можно вычислить сам прямоугольник пересечения.

---

## Зачем это нужно?

- Раскладка интерфейсов (GUI)  
- Обнаружение столкновений в играх и симуляциях  
- Анализ ограничивающих рамок в компьютерном зрении  
- Пространственные запросы в базах данных и ГИС

---

## Как работает алгоритм?

Каждый прямоугольник задаётся двумя точками:  
- Прямоугольник A: (x1_min, y1_min), (x1_max, y1_max)  
- Прямоугольник B: (x2_min, y2_min), (x2_max, y2_max)

### Шаг 1: Проверка на пересечение
- Прямоугольники пересекаются, если:
  - `x1_min < x2_max` и `x1_max > x2_min`
  - `y1_min < y2_max` и `y1_max > y2_min`

### Шаг 2: Вычисление прямоугольника пересечения (опционально)
- Если пересекаются, то:
  - `x_overlap = max(x1_min, x2_min), min(x1_max, x2_max)`
  - `y_overlap = max(y1_min, y2_min), min(y1_max, y2_max)`

---

## Пример на Python

```python
def rectangles_intersect(a, b):
    ax1, ay1, ax2, ay2 = a
    bx1, by1, bx2, by2 = b

    if ax1 >= bx2 or bx1 >= ax2:
        return False
    if ay1 >= by2 or by1 >= ay2:
        return False
    return True

def intersection_rectangle(a, b):
    if not rectangles_intersect(a, b):
        return None
    ax1, ay1, ax2, ay2 = a
    bx1, by1, bx2, by2 = b
    return (
        max(ax1, bx1),
        max(ay1, by1),
        min(ax2, bx2),
        min(ay2, by2)
    )

# Пример использования
rect1 = (1, 1, 5, 5)
rect2 = (3, 3, 6, 6)
print(rectangles_intersect(rect1, rect2))  # True
print(intersection_rectangle(rect1, rect2))  # (3, 3, 5, 5)
```

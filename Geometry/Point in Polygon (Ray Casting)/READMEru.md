# Принадлежность точки многоугольнику (Ray Casting): Объяснение и пример

## Что такое задача Point in Polygon?

Задача **Point in Polygon (PIP)** формулируется так:  
Дан многоугольник и точка. Нужно определить, находится ли точка **внутри**, **снаружи** или **на границе** многоугольника.

---

## Что такое алгоритм Ray Casting?

**Ray Casting** — классический алгоритм для решения задачи PIP.  
Он работает по принципу: из точки проводится горизонтальный луч вправо, и считается, сколько раз он пересекает стороны многоугольника.

- Если число пересечений **нечётное** — точка **внутри**  
- Если **чётное** — точка **снаружи**

---

## Как работает алгоритм Ray Casting?

**Шаг 1: Перебрать все стороны многоугольника**  
- Проверить, пересекает ли луч каждую сторону

**Шаг 2: Подсчитать пересечения**  
- Учитываются только те стороны, которые пересекаются с лучом слева направо

**Шаг 3: Определить результат**  
- Нечётное число пересечений → внутри  
- Чётное → снаружи

---

## Пример на Python

```python
def is_point_in_polygon(point, polygon):
    x, y = point
    n = len(polygon)
    inside = False

    for i in range(n):
        xi, yi = polygon[i]
        xj, yj = polygon[(i + 1) % n]

        if ((yi > y) != (yj > y)):
            x_intersect = (xj - xi) * (y - yi) / (yj - yi + 1e-12) + xi
            if x < x_intersect:
                inside = not inside

    return inside

# Пример использования
polygon = [(0,0), (4,0), (4,4), (0,4)]
point1 = (2,2)
point2 = (5,5)
print(is_point_in_polygon(point1, polygon))  # True
print(is_point_in_polygon(point2, polygon))  # False
```

---

## Сложность
- Время: O(n), где n — число вершин многоугольника
- Память: O(1)

## Где применяется?
- Проверка попадания в область (GUI, игры)

- Обнаружение столкновений
- Анализ форм и областей

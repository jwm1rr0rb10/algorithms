# Ближайшая пара точек: объяснение и пример

## Что такое задача о ближайшей паре точек?

Дано множество точек на плоскости.  
Нужно найти такую пару точек, расстояние между которыми наименьшее среди всех пар.

---

## Как работает алгоритм поиска ближайшей пары точек?

**Грубая сила (brute-force):**  
Проверить расстояния между всеми парами точек.  
- **Сложность:** O(n²)

**Оптимальный метод (разделяй и властвуй):**  
Работает за O(n log n).

**Основная идея:**
1. Отсортировать точки по x-координате.
2. Рекурсивно разбить множество на две части.
3. Найти ближайшую пару в каждой половине.
4. Проверить, нет ли более близкой пары, где одна точка слева, а другая справа (в “полосе” у границы).
5. Вернуть ближайшую пару.

---

## Пример на Python (разделяй и властвуй)

```python
import math

def distance(p1, p2):
    return math.hypot(p1[0] - p2[0], p1[1] - p2[1])

def closest_pair(points):
    points = sorted(points)
    def rec(pts):
        n = len(pts)
        if n <= 3:
            return min((distance(pts[i], pts[j]), (pts[i], pts[j])) for i in range(n) for j in range(i+1, n))
        mid = n // 2
        xmid = pts[mid][0]
        left = pts[:mid]
        right = pts[mid:]
        d1, pair1 = rec(left)
        d2, pair2 = rec(right)
        d, pair = (d1, pair1) if d1 < d2 else (d2, pair2)
        strip = [p for p in pts if abs(p[0] - xmid) < d]
        strip.sort(key=lambda p: p[1])
        for i in range(len(strip)):
            for j in range(i+1, min(i+7, len(strip))):
                d_strip = distance(strip[i], strip[j])
                if d_strip < d:
                    d, pair = d_strip, (strip[i], strip[j])
        return d, pair
    return rec(points)

# Пример использования
points = [(0,0), (2,2), (3,1), (5,5), (1,1)]
print(closest_pair(points))  # Вывод: (1.4142..., ((0,0), (1,1)))
```

---

## Сложность

- **Грубая сила:** O(n²)
- **Разделяй и властвуй:** O(n log n)

---

## Где применяется?

- Компьютерное зрение (поиск похожих объектов, кластеризация)
- Картография и навигация (поиск ближайших точек)
- Робототехника (избежание столкновений)
- Анализ данных (поиск выбросов, пространственные запросы)

---

## Когда использовать алгоритм ближайшей пары точек?

- Когда нужно найти минимальное расстояние между парой точек в наборе
- Когда данных много и важна эффективность

---

## Пример из жизни

Допустим, у вас есть координаты множества сотовых вышек, и нужно найти две ближайшие друг к другу для оптимизации сети.

# Выпуклая оболочка (алгоритм Грэхема): объяснение и пример

## Что такое задача о выпуклой оболочке?

**Выпуклая оболочка** — это наименьший выпуклый многоугольник, содержащий все заданные точки на плоскости.

Представьте, что вы обтягиваете все точки резинкой — она “обхватит” внешний контур: это и есть выпуклая оболочка.

---

## Что такое Graham Scan (алгоритм Грэхема)?

**Graham Scan** — классический эффективный алгоритм построения выпуклой оболочки за O(n log n).

---

## Как работает алгоритм Грэхема?

**Шаг 1: Найти стартовую точку**  
- Ищется точка с минимальной y-координатой (при равенстве — с минимальным x).

**Шаг 2: Отсортировать точки по полярному углу**  
- Остальные точки сортируются по углу относительно стартовой.

**Шаг 3: Проход по точкам (сканирование)**  
- Начинаем обход.
- Для каждой новой точки проверяем: если переход от двух предыдущих к этой точке делает “правый поворот” — предыдущая точка не входит в оболочку, удаляем её. Пока не останется “левый поворот”.
- Добавляем текущую точку.
- Продолжаем так для всех точек.

---

## Пример на Python

```python
def graham_scan(points):
    points = sorted(set(points))  # Удалить дубликаты и отсортировать
    if len(points) <= 1:
        return points

    # Векторное произведение OA x OB
    def cross(o, a, b):
        return (a[0]-o[0])*(b[1]-o[1]) - (a[1]-o[1])*(b[0]-o[0])

    # Нижняя оболочка
    lower = []
    for p in points:
        while len(lower) >= 2 and cross(lower[-2], lower[-1], p) <= 0:
            lower.pop()
        lower.append(p)
    # Верхняя оболочка
    upper = []
    for p in reversed(points):
        while len(upper) >= 2 and cross(upper[-2], upper[-1], p) <= 0:
            upper.pop()
        upper.append(p)
    # Склеить (без повторов)
    return lower[:-1] + upper[:-1]

# Пример использования
points = [(0,0), (1,1), (2,2), (2,0), (2,4), (3,3), (4,2)]
print(graham_scan(points))
# Результат: [(0, 0), (2, 0), (4, 2), (2, 4)]
```

---

## Сложность

- **Время:** O(n log n) (из-за сортировки)
- **Память:** O(n)

---

## Где применяется?

- Компьютерное зрение (обнаружение формы объектов)
- Робототехника (навигация, обход препятствий)
- ГИС (геоинформационные системы)
- Распознавание образов
- Игровая разработка (границы объектов)

---

## Когда использовать выпуклую оболочку?

- Когда нужно построить минимальный выпуклый многоугольник, охватывающий все точки
- Для анализа формы, поиска границ, кластеризации

---

## Пример из жизни

Есть координаты вышек сотовой связи — нужно построить минимальный многоугольник, охватывающий все вышки (выпуклая оболочка).

## Итог

Graham Scan — быстрый и надёжный алгоритм построения выпуклой оболочки, используемый в компьютерном зрении, ГИС и робототехнике.

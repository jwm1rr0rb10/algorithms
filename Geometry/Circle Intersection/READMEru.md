# Пересечение окружностей: объяснение и пример

## Что такое задача на пересечение окружностей?

Дано две окружности (каждая задана центром и радиусом).  
Вопрос: пересекаются ли они? Если да — в каких точках?

Варианты задач:
- Проверить, пересекаются ли окружности (касаются или пересекаются)
- Найти координаты точек пересечения

---

## Как работает алгоритм пересечения окружностей?

Пусть окружности заданы так:
- Окружность 1: центр (x₁, y₁), радиус r₁
- Окружность 2: центр (x₂, y₂), радиус r₂

**Шаг 1: Найти расстояние между центрами**
```
d = sqrt((x₂ - x₁)² + (y₂ - y₁)²)
```

**Шаг 2: Проверить случаи пересечения**
- **Нет пересечения:** d > r₁ + r₂ (слишком далеко)
- **Одна точка (касаются снаружи):** d == r₁ + r₂
- **Две точки (пересекаются):** |r₁ - r₂| < d < r₁ + r₂
- **Одна внутри другой (нет пересечения):** d < |r₁ - r₂|
- **Совпадающие окружности:** бесконечно много точек пересечения

**Шаг 3: Найти точки пересечения (если есть)**

Если окружности пересекаются:

Пусть:
- a = (r₁² - r₂² + d²) / (2d)
- h = sqrt(r₁² - a²)

Тогда точки пересечения:
```
P₀ = (x₁ + a*(x₂ - x₁)/d, y₁ + a*(y₂ - y₁)/d)

(x₃, y₃) = (P₀ₓ ± h*(y₂ - y₁)/d, P₀ᵧ ∓ h*(x₂ - x₁)/d)
```
(“±” — обе точки пересечения, если их две.)

## Пример на Python

```python
import math

def peresechenie_okruzhnostei(x1, y1, r1, x2, y2, r2):
    dx = x2 - x1
    dy = y2 - y1
    d = math.hypot(dx, dy)

    # Нет пересечения
    if d > r1 + r2 or d < abs(r1 - r2):
        return []

    # Совпадающие окружности
    if d == 0 and r1 == r2:
        return ["Бесконечно много точек пересечения (совпадающие окружности)"]

    a = (r1**2 - r2**2 + d**2) / (2 * d)
    h = math.sqrt(max(0, r1**2 - a**2))

    xm = x1 + a * dx / d
    ym = y1 + a * dy / d

    if h == 0:
        return [(xm, ym)]
    else:
        xs1 = xm + h * dy / d
        ys1 = ym - h * dx / d
        xs2 = xm - h * dy / d
        ys2 = ym + h * dx / d
        return [(xs1, ys1), (xs2, ys2)]

# Пример использования:
print(peresechenie_okruzhnostei(0, 0, 5, 8, 0, 5))  # [(4.0, 3.0), (4.0, -3.0)]
```

## Сложность

- **Время:** O(1)
- **Память:** O(1)

---

## Где используется?

- Проверка столкновений в играх и симуляциях
- Анализ перекрытия молекул (химия, биология)
- Робототехника (обход препятствий)
- Компьютерная графика (венновы диаграммы, клиппинг)

---

## Когда использовать пересечение окружностей?

- Если нужно узнать, соприкасаются ли два круглых объекта
- Если нужны координаты точек пересечения для дальнейшей обработки

---

## Пример из жизни

Допустим, у вас есть две Wi-Fi точки с определённым радиусом действия (окружности). Вы хотите узнать, пересекаются ли их зоны покрытия, и если да — где можно разместить устройство, чтобы оно ловило обе сети.

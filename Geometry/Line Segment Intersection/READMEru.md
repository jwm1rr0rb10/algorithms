# Пересечение отрезков: Объяснение и пример

## Что такое задача пересечения отрезков?

Задача **пересечения отрезков** формулируется так:  
Даны два отрезка на плоскости. Нужно определить, пересекаются ли они.  
Если да — найти точку пересечения.

---

## Зачем это нужно?

- Используется в компьютерной графике (отрисовка, отсечение)
- Важна в вычислительной геометрии
- Применяется в ГИС, САПР, робототехнике и играх

---

## Как работает алгоритм?

### Шаг 1: Функция ориентации
- Для трёх точек (p, q, r) определяем ориентацию:
  - 0 → коллинеарны
  - 1 → по часовой стрелке
  - 2 → против часовой стрелки

### Шаг 2: Общий случай
- Отрезки (p1, q1) и (p2, q2) пересекаются, если:
  - ориентации (p1, q1, p2) и (p1, q1, q2) различны
  - и ориентации (p2, q2, p1) и (p2, q2, q1) различны

### Шаг 3: Частные случаи (коллинеарность)
- Проверяем, лежит ли одна из точек на другом отрезке

---

## Пример на Python

```python
def on_segment(p, q, r):
    return min(p[0], r[0]) <= q[0] <= max(p[0], r[0]) and \
           min(p[1], r[1]) <= q[1] <= max(p[1], r[1])

def orientation(p, q, r):
    val = (q[1] - p[1]) * (r[0] - q[0]) - \
          (q[0] - p[0]) * (r[1] - q[1])
    if val == 0:
        return 0  # коллинеарны
    return 1 if val > 0 else 2  # по часовой или против

def do_intersect(p1, q1, p2, q2):
    o1 = orientation(p1, q1, p2)
    o2 = orientation(p1, q1, q2)
    o3 = orientation(p2, q2, p1)
    o4 = orientation(p2, q2, q1)

    if o1 != o2 and o3 != o4:
        return True

    # Частные случаи
    if o1 == 0 and on_segment(p1, p2, q1): return True
    if o2 == 0 and on_segment(p1, q2, q1): return True
    if o3 == 0 and on_segment(p2, p1, q2): return True
    if o4 == 0 and on_segment(p2, q1, q2): return True

    return False

# Пример использования
A = (1, 1)
B = (4, 4)
C = (1, 4)
D = (4, 1)
print(do_intersect(A, B, C, D))  # Вывод: True
```

## Сложность
- Время: O(1) на пару отрезков
- Память: O(1)

## Где применяется?
- Обнаружение столкновений
- Наложение карт в ГИС
- Трассировка лучей
- Планирование маршрутов

## Итог

- Пересечение отрезков — базовая операция в геометрии с широким спектром применения. Метод ориентации эффективен и учитывает как общий, так и частные случаи.

# Выпуклая оболочка (алгоритм Джарвиса): Объяснение и пример

## Что такое задача о выпуклой оболочке?

Задача **выпуклой оболочки** формулируется так:  
Дан набор точек на плоскости. Требуется найти наименьший выпуклый многоугольник, который охватывает все эти точки.

Представьте, что вы натягиваете резинку вокруг точек — она обхватывает их по внешнему контуру. Это и есть выпуклая оболочка.

---

## Что такое алгоритм Джарвиса?

**Алгоритм Джарвиса** (также известен как "заворачивание подарка") — это простой способ построения выпуклой оболочки со сложностью **O(nh)**, где:
- *n* — количество входных точек
- *h* — количество точек на выпуклой оболочке

---

## Как работает алгоритм Джарвиса?

**Шаг 1: Найти самую левую точку**  
- Эта точка гарантированно входит в выпуклую оболочку.

**Шаг 2: Обернуть все точки**  
- Из текущей точки выбираем следующую, которая образует самый левый поворот относительно всех остальных.
- Повторяем, пока не вернёмся в начальную точку.

---

## Пример на Python

```python
def jarvis_march(points):
    if len(points) < 3:
        return points

    hull = []
    leftmost = min(points, key=lambda p: p[0])
    point_on_hull = leftmost

    while True:
        hull.append(point_on_hull)
        endpoint = points[0]
        for p in points[1:]:
            if endpoint == point_on_hull or cross(point_on_hull, endpoint, p) < 0:
                endpoint = p
        point_on_hull = endpoint
        if endpoint == hull[0]:
            break
    return hull

def cross(o, a, b):
    return (a[0]-o[0])*(b[1]-o[1]) - (a[1]-o[1])*(b[0]-o[0])

# Пример использования
points = [(0,0), (1,1), (2,2), (2,0), (2,4), (3,3), (4,2)]
print(jarvis_march(points))
# Вывод: [(0, 0), (2, 0), (4, 2), (2, 4)]
```

## Сложность
- Время: O(nh) — эффективно при небольшом h
- Память: O(h)

## Где применяется?
- Компьютерная графика
- Распознавание образов
- Робототехника
- Геоинформационные системы
- Обнаружение столкновений

## Когда использовать алгоритм Джарвиса?
- Когда число точек на оболочке невелико
- Когда важна простота реализации
- Для обучения и визуализации

## Пример из реальной жизни

- Имея координаты метеостанций, найти минимальную выпуклую область, покрывающую их.

## Итог

- Алгоритм Джарвиса — простой и наглядный способ построения выпуклой оболочки. Он не самый быстрый, но отлично подходит для небольших наборов данных и обучения.

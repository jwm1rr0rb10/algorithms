# Площадь многоугольника: объяснение и пример

## Что такое задача на площадь многоугольника?

Дано: координаты вершин простого многоугольника (невыпуклого или выпуклого, но не самопересекающегося).  
Вопрос: как вычислить его площадь?

---

## Как работает алгоритм площади многоугольника?

Стандартный способ решения — **формула шнуровки** (формула Гаусса).

Для многоугольника с вершинами (x₁, y₁), (x₂, y₂), ..., (xₙ, yₙ), записанными по порядку (по или против часовой стрелки):

```
Площадь = 0.5 * |(x₁y₂ + x₂y₃ + ... + xₙy₁) - (y₁x₂ + y₂x₃ + ... + yₙx₁)|
```

**Пошагово:**
1. Впишите все вершины по порядку, первую повторите в конце.
2. Перемножьте x каждой вершины на y следующей и сложите эти произведения.
3. Перемножьте y каждой вершины на x следующей и сложите эти произведения.
4. Из первой суммы вычтите вторую.
5. Возьмите модуль и разделите на 2.

---

## Пример на Python

```python
def ploshad_mnogoUgolnika(vershiny):
    """
    vershiny: список кортежей (x, y), упорядоченных по многоугольнику.
    """
    n = len(vershiny)
    s = 0
    for i in range(n):
        x1, y1 = vershiny[i]
        x2, y2 = vershiny[(i + 1) % n]
        s += x1 * y2
        s -= y1 * x2
    return abs(s) / 2

# Пример использования
polygon = [(0, 0), (4, 0), (4, 3)]
print(ploshad_mnogoUgolnika(polygon))  # Вывод: 6.0 (площадь треугольника)
```

---

## Сложность

- **Время:** O(n), где n — число вершин
- **Память:** O(1) (не считая входных данных)

---

## Где используется?

- Компьютерная графика и моделирование
- Геоинформационные системы (ГИС)
- Робототехника и навигация
- Обработка изображений

---

## Когда использовать формулу шнуровки?

- Когда известны координаты всех вершин многоугольника
- Многоугольник простой (не самопересекающийся)
- Нет необходимости разбивать на треугольники

---

## Пример из жизни

Допустим, нужно посчитать площадь участка земли, заданного точками GPS. Формула шнуровки позволяет легко вычислить площадь, просто подставив координаты углов участка.

## Итог

Формула шнуровки — быстрый и простой способ найти площадь простого многоугольника по координатам его вершин.

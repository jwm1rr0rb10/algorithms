# Перекрытие прямоугольников: Объяснение и пример

## Что такое задача Rectangle Overlap?

Задача **перекрытия прямоугольников** формулируется так:  
Даны два прямоугольника (выравненные по осям). Нужно определить, **перекрываются** ли они (т.е. имеют ли общую площадь).

---

## Зачем это нужно?

- Раскладка окон и интерфейсов (GUI)  
- Обнаружение столкновений в играх  
- Фильтрация bounding boxes в компьютерном зрении  
- Пространственные запросы и оптимизация

---

## Как работает алгоритм?

Каждый прямоугольник задаётся двумя точками:  
- Прямоугольник A: (x1_min, y1_min), (x1_max, y1_max)  
- Прямоугольник B: (x2_min, y2_min), (x2_max, y2_max)

### Условие перекрытия:
Прямоугольники **не перекрываются**, если один полностью слева, справа, выше или ниже другого.

Значит, они **перекрываются**, если:

```text
x1_min < x2_max И x1_max > x2_min И
y1_min < y2_max И y1_max > y2_min
```

---

## Pyhton Example 

```python
def is_rectangle_overlap(a, b):
    ax1, ay1, ax2, ay2 = a
    bx1, by1, bx2, by2 = b

    return ax1 < bx2 and ax2 > bx1 and ay1 < by2 and ay2 > by1

# Пример использования
rect1 = (1, 1, 4, 4)
rect2 = (2, 2, 5, 5)
rect3 = (5, 5, 7, 7)

print(is_rectangle_overlap(rect1, rect2))  # True
print(is_rectangle_overlap(rect1, rect3))  # False
```

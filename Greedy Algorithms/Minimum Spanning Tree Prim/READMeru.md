# 🌐 Минимальное остовное дерево — Алгоритм Прима

## 📌 Что такое алгоритм Прима?

**Алгоритм Прима** — это жадный алгоритм для построения **минимального остовного дерева (MST)** в связном, неориентированном, взвешенном графе.  
В отличие от Крускала, который добавляет рёбра, Прим начинает с вершины и постепенно расширяет остов, добавляя минимальное ребро к ещё не включённой вершине.

---

## 🎯 Зачем это нужно?

- Эффективен на плотных графах  
- Не требует сортировки всех рёбер  
- Применяется в проектировании сетей, маршрутизации и кластеризации

---

## ⚙️ Как работает алгоритм?

### Шаг 1: Инициализация
- Начинаем с произвольной вершины  
- Используем приоритетную очередь (min-heap) для отслеживания минимальных рёбер к непосещённым вершинам

### Шаг 2: Расширение остова
- На каждом шаге выбираем ребро с минимальным весом, ведущее к новой вершине  
- Добавляем вершину в остов  
- Обновляем очередь новыми рёбрами из этой вершины

---

## 🧪 Пример на Python (с кучей)

```python
import heapq
from collections import defaultdict

def prim(n, edges):
    graph = defaultdict(list)
    for u, v, w in edges:
        graph[u].append((w, v))
        graph[v].append((w, u))

    visited = [False] * n
    min_heap = [(0, 0)]  # (вес, вершина)
    total_weight = 0
    mst = []

    while min_heap:
        weight, u = heapq.heappop(min_heap)
        if visited[u]:
            continue
        visited[u] = True
        total_weight += weight

        for w, v in graph[u]:
            if not visited[v]:
                heapq.heappush(min_heap, (w, v))
                mst.append((u, v, w))

    return mst, total_weight

# Пример использования
edges = [(0, 1, 1), (0, 2, 4), (1, 2, 3), (1, 3, 2), (2, 3, 5)]
mst, weight = prim(4, edges)
print("MST:", mst)
print("Суммарный вес:", weight)
```

---

## ⏱️ Сложность
- Время: O(E + V log V) — с бинарной кучей
- Память: O(V + E) — для графа и очереди

## 🧭 Применение
- Проектирование и маршрутизация сетей
- Кластеризация в машинном обучении
- Сегментация изображений
- Компоновка электронных схем

## ✅ Итог
- Алгоритм Прима строит MST, начиная с вершины
- Использует приоритетную очередь для выбора минимального ребра
- Эффективнее Крускала на плотных графах

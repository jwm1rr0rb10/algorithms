# 🗓️ Планирование задач с дедлайнами: Объяснение и пример

## 📌 Что такое задача планирования задач?

Задача **планирования задач с дедлайнами** формулируется так:  
Дан набор задач, каждая с дедлайном и прибылью. Нужно запланировать задачи так, чтобы максимизировать суммарную прибыль.  
Каждая задача занимает 1 единицу времени, и в каждый момент можно выполнять только одну задачу.

---

## 🎯 Зачем это нужно?

- Моделирует приоритезацию задач в условиях ограниченного времени  
- Применяется в планировании производства, CPU-расписаниях, управлении проектами  
- Демонстрирует жадную оптимизацию с ограничениями по времени

---

## ⚙️ Как работает алгоритм?

### Шаг 1: Отсортировать задачи по убыванию прибыли  
- Жадный выбор: сначала задачи с максимальной прибылью

### Шаг 2: Назначить временные слоты  
- Для каждой задачи ищем последний доступный слот до дедлайна  
- Если слот свободен — назначаем задачу

---

## 🧪 Пример на Python

```python
def job_scheduling(jobs):
    # jobs: список (id, дедлайн, прибыль)
    jobs.sort(key=lambda x: x[2], reverse=True)

    max_deadline = max(job[1] for job in jobs)
    slots = [None] * max_deadline
    total_profit = 0

    for job_id, deadline, profit in jobs:
        for t in range(min(deadline, max_deadline) - 1, -1, -1):
            if slots[t] is None:
                slots[t] = job_id
                total_profit += profit
                break

    return slots, total_profit

# Пример использования
jobs = [('a', 2, 100), ('b', 1, 19), ('c', 2, 27), ('d', 1, 25), ('e', 3, 15)]
schedule, profit = job_scheduling(jobs)
print("Расписание:", schedule)
print("Суммарная прибыль:", profit)
```

## ⏱️ Сложность
- Время: O(n²) — можно ускорить до O(n log n) с помощью структур непересекающихся множеств
- Память: O(n) — для хранения слотов

# 🌲 Минимальное остовное дерево — Алгоритм Крускала

## 📌 Что такое алгоритм Крускала?

**Алгоритм Крускала** — это жадный алгоритм для построения **минимального остовного дерева (MST)** в связном, неориентированном, взвешенном графе.  
Он строит MST, всегда выбирая ребро с наименьшим весом, которое не образует цикл.

---

## 🎯 Зачем это нужно?

- Гарантирует минимальную суммарную длину остова  
- Эффективен на разреженных графах  
- Применяется в сетевом проектировании, кластеризации и компоновке схем

---

## ⚙️ Как работает алгоритм?

### Шаг 1: Отсортировать все рёбра по весу (по возрастанию)

### Шаг 2: Инициализировать структуру непересекающихся множеств (DSU)
- Каждая вершина — в своём собственном множестве

### Шаг 3: Перебрать рёбра
- Для каждого ребра (u, v):
  - Если u и v в разных множествах → добавить ребро в MST и объединить множества
  - Иначе → пропустить (иначе будет цикл)

---

## 🧪 Пример на Python

```python
class DSU:
    def __init__(self, n):
        self.parent = list(range(n))
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])  # Сжатие пути
        return self.parent[x]
    
    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)
        if xr == yr:
            return False
        self.parent[yr] = xr
        return True

def kruskal(n, edges):
    # edges: список (вес, u, v)
    edges.sort()
    dsu = DSU(n)
    mst = []
    total_weight = 0

    for weight, u, v in edges:
        if dsu.union(u, v):
            mst.append((u, v, weight))
            total_weight += weight

    return mst, total_weight

# Пример использования
edges = [(1, 0, 1), (4, 0, 2), (3, 1, 2), (2, 1, 3), (5, 2, 3)]
mst, weight = kruskal(4, edges)
print("MST:", mst)
print("Суммарный вес:", weight)
```

---

## ⏱️ Сложность
- Время: O(E log E) — сортировка рёбер + операции DSU
- Память: O(E) — для хранения рёбер и структуры DSU

---

## 🧭 Применение
- Проектирование сетей (телеком, энергосети)
- Кластеризация в машинном обучении
- Сегментация изображений
- Приближённые алгоритмы (например, TSP)

## ✅ Итог
- Алгоритм Крускала строит MST, жадно выбирая минимальные рёбра
- Использует DSU для эффективного обнаружения циклов
- Идеален для разреженных графов и прост в реализации

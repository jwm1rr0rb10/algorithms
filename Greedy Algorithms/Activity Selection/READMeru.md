# 🧠 Задача о выборе активности: Объяснение и пример

## 📌 Что такое задача о выборе активности?

Задача **о выборе активности** формулируется так:  
Дан список активностей с временем начала и окончания. Необходимо выбрать максимальное количество непересекающихся активностей, которые может выполнить один человек или ресурс.

---

## 🎯 Зачем это нужно?

- Моделирует реальные задачи планирования (например, бронирование переговорных, планирование задач процессора)  
- Демонстрирует силу жадных алгоритмов  
- Лежит в основе задач интервалов и распределения ресурсов

---

## ⚙️ Как работает алгоритм?

### Шаг 1: Отсортировать активности по времени окончания

- Жадный выбор: всегда выбираем активность, которая заканчивается раньше всех

### Шаг 2: Выбрать совместимые активности

- Инициализируем переменную окончания последней выбранной активности  
- Проходим по отсортированному списку:
- Если начало текущей активности ≥ окончание предыдущей → выбираем её

---

## 🧪 Пример на Python

```python
def activity_selection(activities):
    # Сортировка по времени окончания
    activities.sort(key=lambda x: x[1])
    
    selected = []
    last_end = 0

    for start, end in activities:
        if start >= last_end:
            selected.append((start, end))
            last_end = end

    return selected

# Пример использования
activities = [(1, 3), (2, 5), (4, 6), (6, 7), (5, 9), (8, 9)]
print(activity_selection(activities))
# Вывод: [(1, 3), (4, 6), (6, 7), (8, 9)]
```

---

## ⏱️ Сложность
- Время: O(n log n) — из-за сортировки
- Память: O(1) — если не сохранять результат, или O(n) — если сохраняем список выбранных активностей

- Разрешение конфликтов во временных интервалах

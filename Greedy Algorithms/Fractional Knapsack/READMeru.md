# 🧮 Дробный рюкзак (Fractional Knapsack): Объяснение и пример

## 📌 Что такое задача дробного рюкзака?

Задача **дробного рюкзака** формулируется так:  
Дан набор предметов, каждый с весом и стоимостью, и рюкзак с ограниченной грузоподъёмностью. Нужно определить максимальную суммарную стоимость, которую можно получить, взяв целые или дробные части предметов.

В отличие от задачи 0/1 рюкзака, здесь можно брать части предметов.

---

## 🎯 Зачем это нужно?

- Моделирует реальные задачи распределения ресурсов  
- Демонстрирует оптимальность жадной стратегии  
- Применяется в логистике, финансах, распределении пропускной способности

---

## ⚙️ Как работает алгоритм?

### Шаг 1: Вычислить отношение ценности к весу для каждого предмета  
- Отсортировать предметы по убыванию value/weight

### Шаг 2: Жадно заполнять рюкзак  
- Брать как можно больше из предметов с наибольшим отношением  
- Если предмет помещается полностью — взять его  
- Иначе — взять дробную часть, которая влезает

---

## 🧪 Пример на Python

```python
def fractional_knapsack(capacity, items):
    # items: список (стоимость, вес)
    items.sort(key=lambda x: x[0] / x[1], reverse=True)

    total_value = 0.0
    for value, weight in items:
        if capacity == 0:
            break
        if weight <= capacity:
            total_value += value
            capacity -= weight
        else:
            total_value += value * (capacity / weight)
            break

    return total_value

# Пример использования
items = [(60, 10), (100, 20), (120, 30)]
capacity = 50
print(fractional_knapsack(capacity, items))  # Вывод: 240.0
```

---

## ⏱️ Сложность
- Время: O(n log n) — из-за сортировки
- Память: O(1) — при сортировке на месте

---

## 🧭 Применение
- Распределение ресурсов с делимыми объектами
- Распределение пропускной способности
- Оптимизация портфеля
- Балансировка нагрузки

## ✅ Итог
- В дробном рюкзаке можно брать части предметов
- Жадная стратегия оптимальна благодаря отношению ценности к весу
- Эффективен и широко применим в реальных задачах
